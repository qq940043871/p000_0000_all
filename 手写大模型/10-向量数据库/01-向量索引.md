## 一、索引类型

```
向量索引类型：
  1. FLAT（暴力搜索）
     - O(n)遍历
     - 100%准确
  
  2. IVF（倒排文件）
     - 聚类后搜索
     - 近似准确
  
  3. HNSW（分层可导航小世界）
     - 图索引
     - 高召回高速
  
  4. PQ（乘积量化）
     - 有损压缩
     - 内存占用低
```

## 二、HNSW实现

```python
class HNSWIndex:
    """HNSW索引"""
    
    def __init__(self, dimension: int, m: int = 16, 
                 ef_construction: int = 200):
        self.dimension = dimension
        self.m = m  # 连接数
        self.ef_construction = ef_construction
        self.layers = []  # 每层一个列表
    
    def add(self, vector: np.ndarray, id: int):
        """添加向量"""
        level = self._random_level()
        
        for layer_idx in range(level + 1, len(self.layers)):
            if layer_idx >= len(self.layers):
                self.layers.append({"vectors": {}, "neighbors": defaultdict(set)})
            
            # 搜索最近邻
            entry_point = self._search_layer(vector, layer_idx, ef=1)
            
            # 插入
            self.layers[layer_idx]["vectors"][id] = vector
            self._add_connections(id, vector, layer_idx)
    
    def search(self, query: np.ndarray, k: int = 10, 
               ef: int = 100) -> List[Tuple[int, float]]:
        """搜索最近邻"""
        entry_point = self._search_layer(query, len(self.layers) - 1, ef=1)
        
        for layer_idx in range(len(self.layers) - 2, -1, -1):
            entry_point = self._search_layer(query, layer_idx, ef=ef)
        
        # 在底层搜索k个最近邻
        candidates = self._search_layer(query, 0, ef=k)
        return [(doc_id, self._distance(query, vec)) 
                for doc_id, vec in candidates[:k]]
```
