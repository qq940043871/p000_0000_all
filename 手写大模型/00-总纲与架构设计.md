## 技术选型

| 模块 | 技术选择 | 说明 |
|------|---------|------|
| 模型 | LLaMA / GPT-J / ChatGLM | 开源LLM |
| 推理 | GGML / vLLM | 高性能推理 |
| 分词 | Tiktoken / SentencePiece | BPE分词 |
| 向量库 | Faiss / Milvus | ANN检索 |
| Skill | Plugin System | 插件化扩展 |
| MCP | Model Context Protocol | 上下文协议 |
| 框架 | LangChain / LlamaIndex | RAG框架 |
