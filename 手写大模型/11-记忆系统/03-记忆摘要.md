## 一、自动摘要

```python
class MemorySummarizer:
    """记忆摘要器"""
    
    def __init__(self, llm: LLM):
        self.llm = llm
    
    async def summarize(self, messages: List[Dict]) -> str:
        prompt = f"""总结以下对话的要点：

{self._format_messages(messages)}

摘要："""
        return await self.llm.generate(prompt)
```
