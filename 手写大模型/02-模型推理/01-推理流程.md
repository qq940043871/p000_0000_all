## 一、推理步骤

```
推理流程：
  1. 输入编码
     - 文本 → Token IDs
     - 添加特殊Token
  
  2. 模型前向
     - Embedding查询
     - 多层Transformer
     - Self-Attention计算
     - Feed-Forward计算
  
  3. 隐藏状态
     - 最后一层隐藏状态
  
  4. 词汇投影
     - 隐藏状态 → Logits
  
  5. 采样
     - Softmax归一化
     - Top-K/Top-P采样
     - 贪婪采样
  
  6. 解码
     - Token IDs → 文本
```

## 二、实现

```python
class TransformerInference:
    """Transformer推理"""
    
    def __init__(self, model: TransformerModel, config: ModelConfig):
        self.model = model
        self.config = config
        self.kv_cache = None
    
    def forward(self, input_ids: Tensor) -> Tensor:
        # 1. Embedding
        hidden_states = self.model.embed_tokens(input_ids)
        hidden_states = hidden_states * (self.model.config.hidden_size ** 0.5)
        
        # 2. Transformer层
        for layer in self.model.layers:
            hidden_states = self._transformer_layer(layer, hidden_states)
        
        # 3. 最终归一化
        hidden_states = self.model.final_layernorm(hidden_states)
        
        # 4. 词汇投影
        logits = self.model.lm_head(hidden_states)
        
        return logits
    
    def generate(self, input_ids: Tensor, **kwargs) -> Tensor:
        max_length = kwargs.get("max_length", 100)
        temperature = kwargs.get("temperature", 1.0)
        
        for _ in range(max_length):
            logits = self.forward(input_ids)
            logits = logits[:, -1, :] / temperature
            
            # 采样
            probs = F.softmax(logits, dim=-1)
            next_token = torch.multinomial(probs, 1)
            
            input_ids = torch.cat([input_ids, next_token], dim=-1)
            
            # 停止条件
            if next_token.item() == self.eos_token_id:
                break
        
        return input_ids
```
