# 9.1 指标采集

## 概述

指标采集模块负责收集消息中间件运行时的各种性能指标，为监控告警和性能分析提供数据基础。

## 数据库设计

### 表结构：metric_config（指标配置表）

```sql
CREATE TABLE metric_config (
    metric_id          VARCHAR(64)       NOT NULL COMMENT '指标ID',
    metric_name        VARCHAR(128)      NOT NULL COMMENT '指标名称',
    metric_group       ENUM('broker', 'queue', 'exchange', 'consumer', 'producer', 'connection', 'disk', 'memory', 'os') 
                      NOT NULL COMMENT '指标分组',
    
    -- 指标定义
    metric_type        ENUM('counter', 'gauge', 'histogram', 'summary') NOT NULL COMMENT '指标类型',
    unit               VARCHAR(32)       DEFAULT NULL COMMENT '单位(msg/s, bytes, ms, %)',
    description        VARCHAR(512)      DEFAULT NULL COMMENT '指标描述',
    
    -- 采集配置
    collect_interval_s INT               NOT NULL DEFAULT 10 COMMENT '采集间隔(秒)',
    enabled            BOOLEAN           NOT NULL DEFAULT TRUE COMMENT '是否启用',
    
    -- Prometheus 标签
    labels             JSON             DEFAULT NULL COMMENT '标签定义',
    
    PRIMARY KEY (metric_id),
    UNIQUE KEY uk_name (metric_name),
    INDEX idx_group (metric_group)
) ENGINE = InnoDB DEFAULT CHARSET = utf8mb4 COLLATE = utf8mb4_unicode_ci
  COMMENT = '指标配置表';
```

### 表结构：metric_data（指标数据表）

```sql
CREATE TABLE metric_data (
    id                 BIGINT            NOT NULL AUTO_INCREMENT COMMENT 'ID',
    metric_name        VARCHAR(128)      NOT NULL COMMENT '指标名称',
    node_name          VARCHAR(128)      DEFAULT NULL COMMENT '节点名称',
    
    -- 维度
    labels_json        JSON             DEFAULT NULL COMMENT '标签(JSON)',
    
    -- 数据
    value              DOUBLE            NOT NULL COMMENT '指标值',
    timestamp          DATETIME(3)       NOT NULL COMMENT '采集时间',
    
    PRIMARY KEY (id),
    UNIQUE KEY uk_metric_node_time (metric_name, node_name, timestamp),
    INDEX idx_metric_time (metric_name, timestamp),
    INDEX idx_node_time (node_name, timestamp)
) ENGINE = InnoDB DEFAULT CHARSET = utf8mb4 COLLATE = utf8mb4_unicode_ci
  COMMENT = '指标时序数据表';
```

## 核心指标列表

### Broker 级别

| 指标名 | 类型 | 说明 |
|--------|------|------|
| broker_messages_in_total | counter | 总入站消息数 |
| broker_messages_out_total | counter | 总出站消息数 |
| broker_bytes_in_total | counter | 总入站字节数 |
| broker_bytes_out_total | counter | 总出站字节数 |
| broker_connections_open | gauge | 打开的连接数 |
| broker_channels_open | gauge | 打开的信道数 |
| broker_queues_count | gauge | 队列总数 |
| broker_consumers_count | gauge | 消费者总数 |

### Queue 级别

| 指标名 | 类型 | 说明 |
|--------|------|------|
| queue_messages_ready | gauge | 就绪消息数 |
| queue_messages_unacked | gauge | 未确认消息数 |
| queue_messages_total | gauge | 消息总数 |
| queue_consumers | gauge | 消费者数 |
| queue_message_rate_in | gauge | 入队速率 |
| queue_message_rate_out | gauge | 出队速率 |

```yaml
# prometheus.yml
scrape_configs:
  - job_name: 'message-broker'

      - targets: ['broker:9090']
    metrics_path: '/metrics'
    scrape_interval: 15s
```
