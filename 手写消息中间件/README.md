## 📌 设计理念

1. **可靠性优先** — 消息不丢、不重、不乱序是底线
2. **分层解耦** — 协议层、路由层、存储层清晰分离
3. **可观测** — 所有关键路径均有埋点，便于监控告警
4. **易扩展** — 核心模型稳定，业务扩展通过插件实现
