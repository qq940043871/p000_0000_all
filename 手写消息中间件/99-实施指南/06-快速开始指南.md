# 快速开始指南

## 环境准备

| 工具 | 版本要求 | 说明 |
|------|---------|------|
| JDK | 17+ | 如选用 Java 技术栈 |
| Go | 1.21+ | 如选用 Go 技术栈 |
| PostgreSQL | 14+ | 元数据存储 |
| Redis | 7.0+ | 可选，缓存/锁 |
| RocksDB | 8.0+ | 可选，消息存储 |
| Docker | 24+ | 本地开发 |

---

## 第一步：初始化数据库

```bash
# 克隆项目
git clone https://github.com/your-org/message-broker.git
cd message-broker

# 进入实施指南目录，执行 SQL 脚本
psql -h localhost -U postgres -d message_broker \
  -f 99-实施指南/01-数据库建表SQL完整脚本.sql
```

---

## 第二步：配置连接

```yaml
# config.yaml
server:
  host: "0.0.0.0"
  port: 5672
  management_port: 15672

database:
  host: "localhost"
  port: 5432
  name: "message_broker"
  username: "postgres"
  password: "your_password"

storage:
  engine: "rocksdb"
  data_dir: "./data/broker"
```

---

## 第三步：创建第一个队列

```bash
curl -X POST http://localhost:15672/api/v1/queues \
  -H "Content-Type: application/json" \
  -u admin:admin \
  -d '{
    "queue_name": "hello.queue",
    "vhost": "/",
    "durable": true,
    "max_length": 10000
  }'
```

---

## 第四步：发送第一条消息

```bash
curl -X POST http://localhost:15672/api/v1/messages \
  -H "Content-Type: application/json" \
  -u admin:admin \
  -d '{
    "exchange": "",
    "routing_key": "hello.queue",
    "properties": {
      "content_type": "text/plain",
      "delivery_mode": 2
    },
    "payload": "SGVsbG8gV29ybGQh"
  }'
```

---

## 第五步：消费消息

使用 AMQP 客户端（以 Python 为例）：

```python
import pika

credentials = pika.PlainCredentials('admin', 'admin')
params = pika.ConnectionParameters(
    host='localhost',
    port=5672,
    virtual_host='/',
    credentials=credentials
)

connection = pika.BlockingConnection(params)
channel = connection.channel()

# 声明队列
channel.queue_declare(queue='hello.queue', durable=True)

# 消费回调
def callback(ch, method, properties, body):
    print(f"收到消息: {body}")
    ch.basic_ack(delivery_tag=method.delivery_tag)

# 开始消费
channel.basic_consume(queue='hello.queue', on_message_callback=callback)
channel.start_consuming()
```
