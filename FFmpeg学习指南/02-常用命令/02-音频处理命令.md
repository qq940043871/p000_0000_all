## 一、音频提取与转换

### 1.1 从视频中提取音频

```bash
ffmpeg -i input.mp4 -vn -c:a copy output.aac

ffmpeg -i input.mp4 -vn -c:a copy output.mp3

ffmpeg -i input.mp4 -vn -acodec libmp3lame -b:a 192k output.mp3

ffmpeg -i input.mp4 -vn -acodec aac -b:a 128k output.m4a
```

### 1.2 音频格式转换

```bash
ffmpeg -i input.wav -c:a libmp3lame -b:a 192k output.mp3

ffmpeg -i input.mp3 -c:a aac -b:a 128k output.m4a

ffmpeg -i input.wav -c:a libopus -b:a 128k output.opus

ffmpeg -i input.mp3 -c:a libvorbis -b:a 128k output.ogg

ffmpeg -i input.wav -c:a flac output.flac

ffmpeg -i input.wav -c:a pcm_s16le output.wav
```

### 1.3 音频编码器对比

| 编码器 | 格式 | 特点 | 适用场景 |
|--------|------|------|----------|
| libmp3lame | MP3 | 兼容性好 | 通用音频 |
| aac | AAC/M4A | 高效压缩 | 流媒体、移动端 |
| libopus | Opus | 低延迟高质量 | 实时通信 |
| libvorbis | OGG | 开源免费 | Web音频 |
| flac | FLAC | 无损压缩 | 音乐存储 |
| pcm_s16le | WAV | 无压缩 | 音频编辑 |

## 二、音频参数调整

### 2.1 码率设置

```bash
ffmpeg -i input.wav -c:a libmp3lame -b:a 320k output.mp3

ffmpeg -i input.wav -c:a libmp3lame -b:a 192k output.mp3

ffmpeg -i input.wav -c:a libmp3lame -b:a 128k output.mp3

ffmpeg -i input.wav -c:a aac -b:a 256k output.m4a

ffmpeg -i input.wav -c:a libopus -b:a 128k output.opus
```

### 2.2 采样率调整

```bash
ffmpeg -i input.wav -ar 44100 output.wav

ffmpeg -i input.wav -ar 48000 output.wav

ffmpeg -i input.wav -ar 22050 output.wav

ffmpeg -i input.wav -ar 16000 output.wav
```

### 2.3 声道处理

```bash
ffmpeg -i input.wav -ac 1 output.wav

ffmpeg -i input.wav -ac 2 output.wav

ffmpeg -i stereo.wav -af "pan=mono|c0=FL" mono.wav

ffmpeg -i stereo.wav -af "pan=mono|c0=0.5*FL+0.5*FR" mono.wav

ffmpeg -i mono.wav -af "aformat=channel_layouts=stereo" stereo.wav
```

## 三、音量调整

### 3.1 基本音量控制

```bash
ffmpeg -i input.mp3 -af "volume=2.0" output.mp3

ffmpeg -i input.mp3 -af "volume=0.5" output.mp3

ffmpeg -i input.mp3 -af "volume=3dB" output.mp3

ffmpeg -i input.mp3 -af "volume=-3dB" output.mp3
```

### 3.2 音量标准化

```bash
ffmpeg -i input.mp3 -af "loudnorm=I=-16:TP=-1.5:LRA=11" output.mp3

ffmpeg -i input.mp3 -af "loudnorm=I=-14:TP=-1:LRA=11" output.mp3

ffmpeg -i input.mp3 -af "dynaudnorm" output.mp3
```

### 3.3 淡入淡出

```bash
ffmpeg -i input.mp3 -af "afade=t=in:st=0:d=5" output.mp3

ffmpeg -i input.mp3 -af "afade=t=out:st=30:d=5" output.mp3

ffmpeg -i input.mp3 -af "afade=t=in:st=0:d=2,afade=t=out:st=28:d=2" output.mp3
```

## 四、音频剪辑

### 4.1 时间裁剪

```bash
ffmpeg -i input.mp3 -ss 00:01:00 -to 00:02:00 -c copy output.mp3

ffmpeg -i input.mp3 -ss 00:01:00 -t 60 -c copy output.mp3

ffmpeg -i input.mp3 -ss 00:01:00 -c copy output.mp3
```

### 4.2 音频拼接

```bash
cat audio_list.txt
file 'part1.mp3'
file 'part2.mp3'
file 'part3.mp3'

ffmpeg -f concat -safe 0 -i audio_list.txt -c copy output.mp3

ffmpeg -i "concat:part1.mp3|part2.mp3|part3.mp3" -c copy output.mp3
```

## 五、音频混合

### 5.1 添加背景音乐

```bash
ffmpeg -i voice.mp3 -i bgm.mp3 -filter_complex "amix=inputs=2:duration=longest:dropout_transition=2" output.mp3

ffmpeg -i voice.mp3 -i bgm.mp3 -filter_complex "[1:a]volume=0.3[bgm];[0:a][bgm]amix=inputs=2:duration=first" output.mp3
```

### 5.2 音频叠加

```bash
ffmpeg -i main.mp3 -i effect.mp3 -filter_complex "amix=inputs=2:duration=longest" output.mp3

ffmpeg -i a1.mp3 -i a2.mp3 -i a3.mp3 -filter_complex "amix=inputs=3:duration=longest" output.mp3
```

### 5.3 视频替换音频

```bash
ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 output.mp4

ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 -shortest output.mp4
```

## 六、音频特效

### 6.1 均衡器

```bash
ffmpeg -i input.mp3 -af "equalizer=f=100:t=q:w=1:g=5" output.mp3

ffmpeg -i input.mp3 -af "equalizer=f=1000:t=q:w=1:g=-3" output.mp3

ffmpeg -i input.mp3 -af "highpass=f=200,lowpass=f=3000" output.mp3
```

### 6.2 混响效果

```bash
ffmpeg -i input.mp3 -af "aecho=0.8:0.9:1000:0.3" output.mp3

ffmpeg -i input.mp3 -af "aecho=0.8:0.88:60:0.4" output.mp3
```

### 6.3 降噪

```bash
ffmpeg -i input.mp3 -af "afftdn=nf=-25" output.mp3

ffmpeg -i input.mp3 -af "highpass=f=200,lowpass=f=3000,afftdn=nf=-25" output.mp3
```

### 6.4 变速变调

```bash
ffmpeg -i input.mp3 -af "atempo=1.5" output.mp3

ffmpeg -i input.mp3 -af "atempo=0.8" output.mp3

ffmpeg -i input.mp3 -af "asetrate=44100*1.2,aresample=44100" output.mp3

ffmpeg -i input.mp3 -af "rubberband=pitch=1.2" output.mp3
```

## 七、音频分析

### 7.1 查看音频信息

```bash
ffprobe -v quiet -print_format json -show_streams -select_streams a input.mp3

ffprobe -v error -show_entries stream=codec_name,sample_rate,channels,bit_rate input.mp3

ffprobe -v quiet -show_entries format=duration -of default=noprint_wrappers=1:nokey=1 input.mp3
```

### 7.2 音频波形图

```bash
ffmpeg -i input.mp3 -filter_complex "aformat=channel_layouts=mono,showwavespic=s=800x200:colors=667eea" output.png

ffmpeg -i input.mp3 -filter_complex "showwavespic=s=800x200:colors=667eea:split_channels=1" output.png
```

### 7.3 音频频谱图

```bash
ffmpeg -i input.mp3 -filter_complex "showspectrumpic=s=800x400:mode=separate:color=intensity" spectrum.png

ffmpeg -i input.mp3 -filter_complex "showspectrumpic=s=800x400:mode=combined:color=rainbow" spectrum.png
```

## 八、实用脚本

### 8.1 批量提取音频

```bash
#!/bin/bash
for file in *.mp4; do
    ffmpeg -i "$file" -vn -c:a libmp3lame -b:a 192k "${file%.mp4}.mp3"
done
```

### 8.2 批量转换音频格式

```bash
#!/bin/bash
for file in *.wav; do
    ffmpeg -i "$file" -c:a libmp3lame -b:a 192k "${file%.wav}.mp3"
done
```

### 8.3 音频分段

```bash
#!/bin/bash
ffmpeg -i input.mp3 -f segment -segment_time 300 -c copy output_%03d.mp3
```
