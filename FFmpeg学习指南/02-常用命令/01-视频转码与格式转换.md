# 视频转码与格式转换

> 模块：常用命令
> 更新时间：2026-04-06

## 一、基础转码

### 1.1 简单格式转换

```bash
ffmpeg -i input.avi output.mp4

ffmpeg -i input.mov output.mp4

ffmpeg -i input.mp4 output.mkv
```

### 1.2 指定编码器

```bash
ffmpeg -i input.mp4 -c:v libx264 -c:a aac output.mp4

ffmpeg -i input.mp4 -c:v libx265 -c:a aac output.mp4

ffmpeg -i input.mp4 -c:v libvpx-vp9 -c:a libopus output.webm

ffmpeg -i input.mp4 -c:v libaom-av1 -c:a libopus output.mkv
```

### 1.3 保持原编码（快速转换）

```bash
ffmpeg -i input.mp4 -c copy output.mkv

ffmpeg -i input.mov -c copy output.mp4
```

## 二、视频编码参数

### 2.1 H.264编码

```bash
ffmpeg -i input.mp4 -c:v libx264 -preset medium -crf 23 -c:a aac output.mp4
```

**preset参数**（编码速度与压缩率平衡）：
| 参数 | 速度 | 压缩率 | 适用场景 |
|------|------|--------|----------|
| ultrafast | 最快 | 最低 | 实时编码 |
| superfast | 很快 | 低 | 低延迟场景 |
| veryfast | 快 | 中低 | 直播推流 |
| faster | 较快 | 中 | 一般直播 |
| fast | 快 | 中 | 快速转码 |
| medium | 中等 | 中高 | 通用场景 |
| slow | 慢 | 高 | 高质量存储 |
| slower | 很慢 | 很高 | 影视后期 |
| veryslow | 最慢 | 最高 | 最终输出 |

**crf参数**（质量与文件大小）：
| CRF值 | 质量 | 文件大小 |
|-------|------|----------|
| 18-22 | 高质量 | 大 |
| 23 | 默认 | 中等 |
| 24-28 | 中等质量 | 较小 |
| 28+ | 低质量 | 小 |

### 2.2 H.265/HEVC编码

```bash
ffmpeg -i input.mp4 -c:v libx265 -preset medium -crf 28 -c:a aac output.mp4

ffmpeg -i input.mp4 -c:v libx265 -x265-params "keyint=60:min-keyint=60" -crf 28 output.mp4
```

### 2.3 VP9编码

```bash
ffmpeg -i input.mp4 -c:v libvpx-vp9 -crf 30 -b:v 0 -c:a libopus output.webm

ffmpeg -i input.mp4 -c:v libvpx-vp9 -b:v 2M -c:a libopus -b:a 128k output.webm
```

## 三、分辨率与帧率

### 3.1 调整分辨率

```bash
ffmpeg -i input.mp4 -vf "scale=1920:1080" output.mp4

ffmpeg -i input.mp4 -vf "scale=1280:720" output.mp4

ffmpeg -i input.mp4 -vf "scale=854:480" output.mp4

ffmpeg -i input.mp4 -vf "scale=-2:720" output.mp4

ffmpeg -i input.mp4 -vf "scale=1280:-2" output.mp4
```

### 3.2 常用分辨率预设

```bash
ffmpeg -i input.mp4 -vf "scale=3840:2160" output_4k.mp4

ffmpeg -i input.mp4 -vf "scale=1920:1080" output_1080p.mp4

ffmpeg -i input.mp4 -vf "scale=1280:720" output_720p.mp4

ffmpeg -i input.mp4 -vf "scale=640:360" output_360p.mp4
```

### 3.3 调整帧率

```bash
ffmpeg -i input.mp4 -r 30 output.mp4

ffmpeg -i input.mp4 -r 24 output.mp4

ffmpeg -i input.mp4 -vf "fps=25" output.mp4

ffmpeg -i input.mp4 -vf "fps=film" output.mp4
```

## 四、码率控制

### 4.1 固定码率(CBR)

```bash
ffmpeg -i input.mp4 -b:v 4M -b:a 128k -minrate 4M -maxrate 4M -bufsize 8M output.mp4
```

### 4.2 可变码率(VBR)

```bash
ffmpeg -i input.mp4 -b:v 4M -maxrate 6M -bufsize 8M output.mp4
```

### 4.3 平均码率(ABR)

```bash
ffmpeg -i input.mp4 -b:v 4M output.mp4
```

### 4.4 质量控制(CRF)

```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 23 output.mp4
```

### 4.5 两遍编码

```bash
ffmpeg -i input.mp4 -c:v libx264 -b:v 4M -pass 1 -f null /dev/null
ffmpeg -i input.mp4 -c:v libx264 -b:v 4M -pass 2 output.mp4
```

## 五、视频裁剪与拼接

### 5.1 时间裁剪

```bash
ffmpeg -i input.mp4 -ss 00:01:00 -to 00:02:00 -c copy output.mp4

ffmpeg -i input.mp4 -ss 00:01:00 -t 60 -c copy output.mp4

ffmpeg -ss 00:01:00 -i input.mp4 -t 60 -c copy output.mp4
```

### 5.2 画面裁剪

```bash
ffmpeg -i input.mp4 -vf "crop=1920:1080:0:0" output.mp4

ffmpeg -i input.mp4 -vf "crop=iw:ih-120:0:60" output.mp4

ffmpeg -i input.mp4 -vf "crop=1280:720:320:180" output.mp4

ffmpeg -i input.mp4 -vf "cropdetect" -f null - 2>&1 | grep crop
```

### 5.3 视频拼接

```bash
cat filelist.txt
file 'part1.mp4'
file 'part2.mp4'
file 'part3.mp4'

ffmpeg -f concat -safe 0 -i filelist.txt -c copy output.mp4

ffmpeg -f concat -safe 0 -i filelist.txt -c:v libx264 -c:a aac output.mp4
```

## 六、滤镜处理

### 6.1 常用滤镜

```bash
ffmpeg -i input.mp4 -vf "hflip" output.mp4

ffmpeg -i input.mp4 -vf "vflip" output.mp4

ffmpeg -i input.mp4 -vf "transpose=1" output.mp4

ffmpeg -i input.mp4 -vf "rotate=45*PI/180" output.mp4

ffmpeg -i input.mp4 -vf "eq=brightness=0.1:contrast=1.2" output.mp4

ffmpeg -i input.mp4 -vf "eq=saturation=1.5" output.mp4

ffmpeg -i input.mp4 -vf "unsharp=5:5:1.0:5:5:0.0" output.mp4

ffmpeg -i input.mp4 -vf "boxblur=2:1" output.mp4
```

### 6.2 添加水印

```bash
ffmpeg -i input.mp4 -i logo.png -filter_complex "overlay=10:10" output.mp4

ffmpeg -i input.mp4 -i logo.png -filter_complex "overlay=W-w-10:10" output.mp4

ffmpeg -i input.mp4 -i logo.png -filter_complex "overlay=W-w-10:H-h-10" output.mp4

ffmpeg -i input.mp4 -i logo.png -filter_complex "overlay=10:H-h-10" output.mp4

ffmpeg -i input.mp4 -i logo.png -filter_complex "overlay=(W-w)/2:(H-h)/2" output.mp4
```

### 6.3 添加文字

```bash
ffmpeg -i input.mp4 -vf "drawtext=text='Hello World':fontsize=24:fontcolor=white:x=10:y=10" output.mp4

ffmpeg -i input.mp4 -vf "drawtext=text='Hello':fontsize=48:fontcolor=white:x=(w-text_w)/2:y=(h-text_h)/2" output.mp4

ffmpeg -i input.mp4 -vf "drawtext=text='%{localtime\:%Y-%m-%d %H\\\\\:%M\\\\\:%S}':fontsize=24:fontcolor=white:x=10:y=10" output.mp4
```

## 七、格式转换实战

### 7.1 转换为Web格式

```bash
ffmpeg -i input.mp4 -c:v libx264 -preset slow -crf 22 -c:a aac -b:a 128k -movflags +faststart output.mp4
```

### 7.2 转换为HLS

```bash
ffmpeg -i input.mp4 -c:v libx264 -c:a aac -f hls -hls_time 10 -hls_list_size 0 -hls_segment_filename "segment_%03d.ts" playlist.m3u8
```

### 7.3 转换为DASH

```bash
ffmpeg -i input.mp4 -c:v libx264 -c:a aac -f dash -seg_duration 10 -init_seg_name "init.m4s" -media_seg_name "segment_%d.m4s" manifest.mpd
```

### 7.4 生成GIF

```bash
ffmpeg -i input.mp4 -ss 00:00:05 -t 3 -vf "fps=10,scale=480:-1:flags=lanczos" output.gif

ffmpeg -i input.mp4 -vf "fps=15,scale=320:-1:flags=lanczos,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse" output.gif
```

## 八、批量处理脚本

### 8.1 批量转码

```bash
#!/bin/bash
for file in *.avi; do
    ffmpeg -i "$file" -c:v libx264 -preset medium -crf 23 -c:a aac "${file%.avi}.mp4"
done
```

### 8.2 批量调整分辨率

```bash
#!/bin/bash
for file in *.mp4; do
    ffmpeg -i "$file" -vf "scale=1280:720" -c:v libx264 -preset fast -c:a copy "720p_${file}"
done
```
