## 一、符号表结构

```java
public class SymbolTable {
    private Map<String, Symbol> symbols = new HashMap<>();
    private Map<String, TableInfo> tables = new HashMap<>();
    private SymbolTable parent;
    
    public SymbolTable() {
        this(null);
    }
    
    public SymbolTable(SymbolTable parent) {
        this.parent = parent;
    }
    
    // 作用域管理
    public SymbolTable createChildScope() {
        return new SymbolTable(this);
    }
    
    public SymbolTable getParentScope() {
        return parent;
    }
    
    // 符号管理
    public void defineSymbol(String name, Symbol symbol) {
        symbols.put(name, symbol);
    }
    
    public Symbol resolveSymbol(String name) {
        Symbol symbol = symbols.get(name);
        if (symbol != null) return symbol;
        if (parent != null) return parent.resolveSymbol(name);
        return null;
    }
    
    // 表管理
    public void defineTable(TableInfo table) {
        tables.put(table.getName(), table);
    }
    
    public TableInfo resolveTable(String name) {
        TableInfo table = tables.get(name);
        if (table != null) return table;
        if (parent != null) return parent.resolveTable(name);
        return null;
    }
}
```

public class Symbol {
    private String name;

    public enum SymbolType {
        COLUMN,
        VARIABLE,
        PARAMETER,
        FUNCTION
    }
}

public class TableInfo {
    private String name;
    private List<ColumnInfo> columns;
    private Map<String, Integer> columnIndex;
    private IndexInfo primaryKey;
    
    public ColumnInfo getColumn(String name) {
        Integer idx = columnIndex.get(name.toLowerCase());
        return idx != null ? columns.get(idx) : null;
    }
}

public class ColumnInfo {
    private String name;
    private DataType type;
    private boolean nullable;
    private boolean primaryKey;
    private boolean autoIncrement;
    private String defaultValue;
}

## 三、类型定义

```java
public enum DataType {
    INT,
    BIGINT,
    SMALLINT,
    TINYINT,
    FLOAT,
    DOUBLE,
    DECIMAL,
    CHAR,
    VARCHAR,
    TEXT,
    BLOB,
    DATE,
    DATETIME,
    TIMESTAMP,
    BOOLEAN
}

public class ColumnDefinition {
    private String name;
    private DataType type;
    private int length;
    private int precision;
    private int scale;
    private boolean nullable = true;
    private boolean primaryKey = false;
    private boolean autoIncrement = false;
    private Object defaultValue;
    
    public DataType inferType() {
        switch (type) {
            case INT:
            case INTEGER:
                return DataType.INT;
            case VARCHAR:
                return DataType.VARCHAR;
            case TEXT:
                return DataType.TEXT;
            default:
                return type;
        }
    }
}
```

## 四、作用域链

```
全局作用域（Global Scope）
├── 数据库: mydb
│   ├── 表: users
│   │   ├── 列: id (INT, PK)
│   │   ├── 列: name (VARCHAR)
│   │   └── 列: age (INT)
│   └── 表: orders
│       ├── 列: id (INT, PK)
│       ├── 列: user_id (INT, FK)
│       └── 列: amount (DECIMAL)
│
查询作用域（Query Scope）
├── 别名: u (users)
└── 别名: o (orders)
```
