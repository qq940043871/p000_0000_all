## 一、崩溃恢复流程

```
崩溃恢复阶段：
  1. 分析阶段（Analysis）
     - 确定数据库崩溃时的状态
     - 确定需要恢复的事务
  
  2. 重做阶段（Redo）
     - 从检查点开始重做所有操作
     - 保证已提交事务的持久性
  
  3. 回滚阶段（Undo）
     - 回滚未提交事务
     - 保证原子性
```

## 二、完整恢复流程

```java
public class CrashRecovery {
    private LogManager logManager;
    private BufferPool bufferPool;
    private TransactionManager txManager;
    
    public void recover() {
        System.out.println("Starting crash recovery...");
        
        // 1. 分析阶段
        RecoveryAnalysis analysis = analyze();
        System.out.println("Analysis complete: " + 
            analysis.getCommittedTxns().size() + " committed, " +
            analysis.getUncommittedTxns().size() + " uncommitted");
        
        // 2. Redo阶段
        redo(analysis);
        System.out.println("Redo complete");
        
        // 3. Undo阶段
        undo(analysis);
        System.out.println("Undo complete");
        
        System.out.println("Recovery complete");
    }
    
    private RecoveryAnalysis analyze() {
        RecoveryAnalysis analysis = new RecoveryAnalysis();
        
        // 读取日志，找到最后一个检查点
        CheckpointRecord checkpoint = findLastCheckpoint();
        long startLSN = checkpoint != null ? checkpoint.getLSN() : 0;
        
        // 扫描日志，确定事务状态
        for (LogRecord record : logManager.scanFrom(startLSN)) {
            switch (record.getType()) {
                case BEGIN:
                    analysis.addActiveTransaction(record.getTxId());
                    break;
                case COMMIT:
                    analysis.commitTransaction(record.getTxId());
                    break;
                case ROLLBACK:
                    analysis.removeTransaction(record.getTxId());
                    break;
                case UPDATE:
                    analysis.addDirtyPage(record.getPageId());
                    break;
            }
        }
        
        return analysis;
    }
    
    private void redo(RecoveryAnalysis analysis) {
        long redoLSN = analysis.getRedoLSN();
        
        for (LogRecord record : logManager.scanFrom(redoLSN)) {
            if (record.getType() == LogType.UPDATE) {
                if (analysis.shouldRedo(record)) {
                    applyRedo(record);
                }
            }
        }
    }
    
    private void undo(RecoveryAnalysis analysis) {
        for (Long txId : analysis.getUncommittedTxns()) {
            rollbackTransaction(txId);
        }
    }
}
```
