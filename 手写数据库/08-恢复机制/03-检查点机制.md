## 一、检查点原理

```
检查点（Checkpoint）：
  - 将脏页刷新到磁盘
  - 记录恢复起点
  - 减少恢复时间

类型：
  - Sharp Checkpoint：刷新所有脏页
  - Fuzzy Checkpoint：渐进刷新
```

---

## 二、检查点实现

```java
public class CheckpointManager {
    private BufferPool bufferPool;
    private LogManager logManager;
    private long checkpointId;
    
    public void doCheckpoint() {
        synchronized (this) {
            // 1. 记录检查点开始
            long checkpointLSN = logManager.logCheckpointStart(checkpointId);
            
            // 2. 收集脏页
            List<DirtyPage> dirtyPages = bufferPool.getDirtyPages();
            
            // 3. 刷新脏页
            for (DirtyPage page : dirtyPages) {
                bufferPool.flushPage(page.getPageId());
            }
            
            // 4. 记录检查点结束
            CheckpointRecord record = new CheckpointRecord();
            record.setCheckpointId(checkpointId);
            record.setCheckpointLSN(checkpointLSN);
            record.setDirtyPages(dirtyPages);
            
            logManager.logCheckpointEnd(record);
            
            checkpointId++;
        }
    }
}
```

## 三、增量检查点

```java
public class IncrementalCheckpointManager {
    private static final int MIN_CHECKPOINT_INTERVAL = 5; // 秒
    private static final int MAX_DIRTY_PAGES = 1000;
    
    public void checkAndCheckpoint() {
        long now = System.currentTimeMillis();
        long elapsed = now - lastCheckpointTime;
        
        List<DirtyPage> dirtyPages = bufferPool.getDirtyPages();
        
        if (elapsed > MIN_CHECKPOINT_INTERVAL * 1000 || 
            dirtyPages.size() > MAX_DIRTY_PAGES) {
            doIncrementalCheckpoint(dirtyPages);
            lastCheckpointTime = now;
        }
    }
}
```
