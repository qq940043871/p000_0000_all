## 一、主键索引概述

```
主键索引特点：
  - 唯一且非空
  - 自动创建
  - 默认使用B+树
  - 数据按主键排序存储
```

## 二、索引管理

```java
public class IndexManager {
    private Map<String, BPlusTree> indexes = new HashMap<>();
    
    public void createPrimaryKeyIndex(String tableName, List<String> columns) {
        TableSchema schema = catalog.getTableSchema(tableName);
        IndexInfo indexInfo = new IndexInfo();
        indexInfo.setName("PRIMARY");
        indexInfo.setTableName(tableName);
        indexInfo.setColumns(columns);
        indexInfo.setUnique(true);
        
        BPlusTree tree = new BPlusTree();
        indexes.put(tableName + "_PRIMARY", tree);
        
        // 填充索引
        for (Row row : tableScanner.scan(tableName)) {
            byte[] key = encodeKey(row.getPrimaryKey(), schema);
            RowPointer pointer = row.getPointer();
            tree.insert(key, pointer);
        }
    }
    
    public IndexInfo getPrimaryKeyIndex(String tableName) {
        return catalog.getTableSchema(tableName).getPrimaryKeyIndex();
    }
    
    public List<RowPointer> search(String tableName, byte[] key) {
        String indexName = tableName + "_PRIMARY";
        BPlusTree tree = indexes.get(indexName);
        return tree.search(key);
    }
}
```
