# 普通索引

> 模块：索引系统
> 更新时间：2026-03-29

---

## 一、普通索引概述

```
普通索引（Secondary Index）：
  - 允许重复值
  - 可有多个
  - 用于加速查询
```

---

## 二、实现

```java
public class SecondaryIndex extends BPlusTree {
    private String indexName;
    private String tableName;
    private List<String> columns;
    
    public void insert(byte[] key, RowPointer pointer) {
        // 普通索引允许重复，存储所有匹配的行
        super.insert(key, pointer);
    }
    
    public List<RowPointer> search(byte[] key) {
        return super.search(key);
    }
    
    public List<RowPointer> rangeSearch(byte[] startKey, byte[] endKey) {
        return super.rangeSearch(startKey, endKey);
    }
}
```
