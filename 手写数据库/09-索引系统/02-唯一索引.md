# 唯一索引

> 模块：索引系统
> 更新时间：2026-03-29

---

## 一、唯一索引概述

```
唯一索引特点：
  - 允许有空值（但只能有一个）
  - 不允许重复值
  - 可在创建表后添加
```

---

## 二、实现

```java
public class UniqueIndex extends BPlusTree {
    private String tableName;
    private List<String> columns;
    
    @Override
    public void insert(byte[] key, RowPointer pointer) {
        // 检查是否已存在
        List<RowPointer> existing = search(key);
        if (!existing.isEmpty()) {
            throw new DuplicateKeyException("Duplicate key: " + decodeKey(key));
        }
        
        super.insert(key, pointer);
    }
}
```
