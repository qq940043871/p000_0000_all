## 一、页结构

```
InnoDB页结构（16KB）：
┌─────────────────────────────────────────────────────────┐
│  File Header（38字节）                                    │
│  ├─ Checksum (4字节)                                      │
│  ├─ Page Offset (4字节)                                  │
│  ├─ Previous Page (4字节)                                │
│  ├─ Next Page (4字节)                                    │
│  ├─ Page Type (2字节)                                    │
│  └─ ...                                                  │
├─────────────────────────────────────────────────────────┤
│  Page Header（56字节）                                    │
│  ├─ Page Directory Offset                                │
│  ├─ Number of Directory Slots                            │
│  ├─ Heap Top                                              │
│  ├─ Number of Records                                    │
│  └─ ...                                                  │
├─────────────────────────────────────────────────────────┤
│  Infimum + Supremum Records（26字节）                     │
├─────────────────────────────────────────────────────────┤
│  User Records（可变）                                     │
│  ├─ Record 1                                             │
│  ├─ Record 2                                             │
│  └─ ...                                                  │
├─────────────────────────────────────────────────────────┤
│  Free Space（可变）                                       │
├─────────────────────────────────────────────────────────┤
│  Page Directory（可变）                                    │
│  ├─ Slot 1                                               │
│  ├─ Slot 2                                               │
│  └─ ...                                                  │
├─────────────────────────────────────────────────────────┤
│  File Trailer（8字节）                                    │
└─────────────────────────────────────────────────────────┘
```

## 二、Java实现

```java
public class Page {
    public static final int PAGE_SIZE = 16384;  // 16KB
    public static final int HEADER_SIZE = 38;
    public static final int PAGE_HEADER_SIZE = 56;
    
    private byte[] data = new byte[PAGE_SIZE];
    
    // 页类型
    public static final short TYPE_INDEX = 17855;  // 0x45BF
    public static final short TYPE_DATA = 17922;  // 0x45E2
    
    // 获取页类型
    public short getPageType() {
        return ByteBuffer.wrap(data, 34, 2).order(ByteOrder.LITTLE_ENDIAN).getShort();
    }
    
    // 获取记录数
    public int getRecordCount() {
        return ByteBuffer.wrap(data, HEADER_SIZE + 2, 2)
            .order(ByteOrder.LITTLE_ENDIAN).getShort();
    }
    
    // 获取页目录槽数
    public int getDirectorySlots() {
        return ByteBuffer.wrap(data, HEADER_SIZE + 16, 2)
            .order(ByteOrder.LITTLE_ENDIAN).getShort();
    }
}
```

## 四、行操作

```java
public class Row {
    private Object[] columns;
    private long transactionId;
    private long rollPointer;
    private boolean deleted;
    
    public byte[] toBytes(TableSchema schema) {
        ByteArrayOutputStream out = new ByteArrayOutputStream();
        
        // 写入变长字段长度
        List<byte[]> varFields = new ArrayList<>();
        int varLen = 0;
        for (int i = 0; i < columns.length; i++) {
            if (schema.getColumn(i).isVariableLength()) {
                byte[] fieldData = encodeValue(columns[i]);
                varFields.add(fieldData);
                varLen += fieldData.length;
            }
        }
        
        // 写入NULL位图
        byte[] nullBitmap = buildNullBitmap();
        out.write(nullBitmap);
        
        // 写入变长字段
        for (byte[] field : varFields) {
            out.write(field);
        }
        
        // 写入固定长度字段
        for (int i = 0; i < columns.length; i++) {
            if (!schema.getColumn(i).isVariableLength()) {
                out.write(encodeFixedLength(columns[i], schema.getColumn(i)));
            }
        }
        
        return out.toByteArray();
    }
}
```
