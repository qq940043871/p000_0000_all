## 一、缓冲池结构

```
缓冲池作用：
  - 缓存磁盘数据页
  - 减少磁盘IO
  - 提高查询性能

核心组件：
  - 页缓存：LRU链表
  - 脏页管理：标记修改过的页
  - 淘汰策略：LRU/KnowledgeTree
```

## 二、缓冲池实现

```java
public class BufferPool {
    private int poolSize;  // 页数
    private Map<Long, BufferFrame> pageTable;  // 页ID → 帧
    private LinkedHashMap<Long, BufferFrame> lruList;  // LRU链表
    private DiskManager diskManager;
    
    public BufferPool(int poolSize, DiskManager diskManager) {
        this.poolSize = poolSize;
        this.diskManager = diskManager;
        this.pageTable = new ConcurrentHashMap<>();
        this.lruList = new LinkedHashMap<>(poolSize, 0.75f, true) {
            @Override
            protected boolean removeEldestEntry(Map.Entry<Long, BufferFrame> eldest) {
                return size() > poolSize;
            }
        };
    }
    
    // 获取页
    public BufferFrame getPage(long pageId) {
        BufferFrame frame = pageTable.get(pageId);
        
        if (frame == null) {
            // 页面不在缓冲池，需要加载
            frame = loadPage(pageId);
        }
        
        // 移到LRU头部
        lruList.remove(pageId);
        lruList.put(pageId, frame);
        
        frame.pin();  // 增加引用计数
        return frame;
    }
    
    private BufferFrame loadPage(long pageId) {
        // 如果缓冲池已满，淘汰一页
        if (pageTable.size() >= poolSize) {
            evictPage();
        }
        
        // 从磁盘读取
        byte[] data = diskManager.readPage(pageId);
        BufferFrame frame = new BufferFrame(pageId, data);
        
        pageTable.put(pageId, frame);
        lruList.put(pageId, frame);
        
        return frame;
    }
    
    // 淘汰页
    private void evictPage() {
        Long oldestPageId = lruList.keySet().iterator().next();
        BufferFrame frame = lruList.remove(oldestPageId);
        
        if (frame.isDirty()) {
            diskManager.writePage(frame.getPageId(), frame.getData());
        }
        
        pageTable.remove(frame.getPageId());
    }
    
    // 释放页
    public void unpinPage(BufferFrame frame) {
        frame.unpin();
    }
}
```

## 三、LRU-K实现

```java
public class LRU_KBufferPool extends BufferPool {
    private int k = 2;  // 记录最近K次访问
    
    @Override
    public void unpinPage(BufferFrame frame) {
        super.unpinPage(frame);
        
        // 记录访问历史
        frame.addAccessTime(System.currentTimeMillis());
        
        // 如果访问次数超过K，评估是否淘汰
        if (frame.getAccessCount() > k) {
            long backwardK = frame.getBackwardKDistance();
            if (backwardK > threshold) {
                evict(frame);
            }
        }
    }
}
```
