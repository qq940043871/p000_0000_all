## 技术选型

| 模块 | 技术选择 | 说明 |
|------|---------|------|
| 语言 | Java | 面向对象，跨平台 |
| 存储 | 自定义文件格式 | 类MySQL InnoDB页结构 |
| 索引 | B+树 | 磁盘索引结构 |
| 事务 | MVCC + 锁 | 隔离级别支持 |
| 日志 | WAL | Redo + Undo |
| 网络 | Java NIO | 高并发连接 |
