## 一、场景图

```
场景图（Scene Graph）：
  - 层次结构表示游戏对象关系
  - 变换继承（世界坐标计算）
  - 视锥裁剪优化
  - 渲染排序

节点类型：
  - 变换节点（Transform）
  - 渲染节点（Mesh）
  - 灯光节点（Light）
  - 相机节点（Camera）
  - 分组节点（Group）
```

## 二、节点结构

```cpp
class SceneNode {
private:
    std::string name;
    SceneNode* parent = nullptr;
    std::vector<SceneNode*> children;
    
    Transform transform;
    Matrix4x4 worldMatrix;
    bool worldMatrixDirty = true;
    
public:
    void addChild(SceneNode* child) {
        child->parent = this;
        children.push_back(child);
    }
    
    void removeChild(SceneNode* child) {
        children.erase(
            std::remove(children.begin(), children.end(), child),
            children.end()
        );
        child->parent = nullptr;
    }
    
    const Matrix4x4& getWorldMatrix() {
        if (worldMatrixDirty) {
            updateWorldMatrix();
        }
        return worldMatrix;
    }
    
private:
    void updateWorldMatrix() {
        if (parent) {
            worldMatrix = parent->getWorldMatrix() * transform.getLocalMatrix();
        } else {
            worldMatrix = transform.getLocalMatrix();
        }
        worldMatrixDirty = false;
        
        for (auto* child : children) {
            child->worldMatrixDirty = true;
        }
    }
};
```
