## 技术选型

| 模块 | 技术选择 | 说明 |
|------|---------|------|
| 语言 | C++ | 性能首选 |
| 脚本 | Lua/Python | 游戏逻辑 |
| 渲染 | OpenGL/Vulkan | 跨平台图形 |
| 物理 | 自研/Bullet | 刚体碰撞 |
| 音频 | OpenAL/FMOD | 3D音频 |
| 数学库 | GLM/Eigen | 向量矩阵 |
