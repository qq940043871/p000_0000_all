## 一、内存优化策略

```
优化策略：
  1. 对象池
     - 预分配常用对象
     - 减少new/delete调用
  
  2. 资源流式加载
     - 按需加载资源
     - 异步加载
  
  3. 内存池
     - 固定大小分配
     - 减少碎片
  
  4. 数据压缩
     - 顶点数据压缩
     - 纹理压缩（ASTC/BC/DXT）
```

## 二、内存分析

```cpp
class MemoryProfiler {
private:
    std::unordered_map<void*, AllocationInfo> allocations;
    std::mutex mutex;
    
public:
    void onAllocate(void* ptr, size_t size, const char* file, int line) {
        std::lock_guard<std::mutex> lock(mutex);
        
        AllocationInfo info;
        info.size = size;
        info.file = file;
        info.line = line;
        info.timestamp = getTime();
        
        allocations[ptr] = info;
    }
    
    void onDeallocate(void* ptr) {
        std::lock_guard<std::mutex> lock(mutex);
        allocations.erase(ptr);
    }
    
    void report() {
        size_t totalSize = 0;
        std::vector<AllocationInfo*> allocs;
        
        for (auto& pair : allocations) {
            totalSize += pair.second.size;
            allocs.push_back(&pair.second);
        }
        
        // 按大小排序
        std::sort(allocs.begin(), allocs.end(), 
            [](auto* a, auto* b) { return a->size > b->size; });
        
        LOG("Total Allocations: %zu, Size: %zu bytes", 
            allocations.size(), totalSize);
        
        for (size_t i = 0; i < 20 && i < allocs.size(); i++) {
            LOG("  %s:%d - %zu bytes", 
                allocs[i]->file, allocs[i]->line, allocs[i]->size);
        }
    }
};
```
