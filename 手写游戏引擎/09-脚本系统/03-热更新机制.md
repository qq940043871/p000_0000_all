# 热更新机制

> 模块：脚本系统
> 更新时间：2026-03-29

---

## 一、热更新流程

```
1. 检测文件变化
2. 备份原脚本
3. 加载新脚本
4. 重新初始化脚本状态
5. 恢复引用关系
6. 记录更新日志
```

---

## 二、实现

```cpp
class HotReloadManager {
private:
    std::unordered_map<std::string, FileWatcher*> watchers;
    std::unordered_map<std::string, LuaScript*> scripts;
    
public:
    void watch(const std::string& path) {
        FileWatcher* watcher = new FileWatcher(path);
        watcher->onChange([this, path]() {
            reloadScript(path);
        });
        watchers[path] = watcher;
    }
    
    void reloadScript(const std::string& path) {
        auto it = scripts.find(path);
        if (it != scripts.end()) {
            LuaScript* oldScript = it->second;
            
            // 保存状态
            ScriptState state = oldScript->captureState();
            
            // 重新加载
            oldScript->load(path);
            
            // 恢复状态
            oldScript->restoreState(state);
            
            // 触发回调
            oldScript->onReload();
        }
    }
};
```
