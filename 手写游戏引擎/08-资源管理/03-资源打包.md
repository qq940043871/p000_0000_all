## 一、资源包格式

```
ResourcePack格式：
┌────────────────────────────────┐
│  Header (16 bytes)              │
│  - Magic: "RPK1"               │
│  - Version                     │
│  - Entry Count                 │
│  - String Table Offset         │
├────────────────────────────────┤
│  Entry Table                    │
│  - Path Hash                   │
│  - Offset                      │
│  - Compressed Size             │
│  - Raw Size                    │
│  - Compression Type            │
├────────────────────────────────┤
│  Data Blocks                    │
│  - Compressed Data              │
└────────────────────────────────┘
```

## 二、打包工具

```cpp
class ResourcePacker {
public:
    void pack(const std::string& inputDir, const std::string& outputPath) {
        std::ofstream out(outputPath, std::ios::binary);
        
        // 收集所有资源
        std::vector<FileEntry> entries;
        collectFiles(inputDir, entries);
        
        // 写入头部
        writeHeader(out, entries);
        
        // 写入数据
        for (auto& entry : entries) {
            compressAndWrite(out, entry);
        }
    }
    
private:
    void compressAndWrite(std::ofstream& out, FileEntry& entry) {
        std::vector<uint8_t> data = readFile(entry.rawPath);
        
        // 压缩
        std::vector<uint8_t> compressed;
        if (shouldCompress(entry.extension)) {
            compressed = gzipCompress(data);
            entry.compressionType = CompressionType::GZIP;
        } else {
            compressed = data;
            entry.compressionType = CompressionType::NONE;
        }
        
        entry.compressedSize = compressed.size();
        entry.offset = out.tellp();
        out.write((char*)compressed.data(), compressed.size());
    }
};
```
