# 豆包AI视频生成 · 剧情连贯技巧

> 目标：用提示词控制豆包，生成剧情连贯的视频，而不是一堆碎片。

---

## 一、核心原则：分段生成 + 统一设定

```
豆包的工作方式：
  - 一次生成一个场景（几秒到十几秒）
  - 无法一次生成长剧情

正确策略：
  1. 先规划完整剧情（分镜脚本）
  2. 统一角色/场景设定
  3. 分段生成每个场景
  4. 后期剪辑拼接
```

---

## 二、第一步：建立统一设定

### 角色设定模板

```
【角色A】
- 外貌：25岁女性，黑色长发，戴细框眼镜，穿白色衬衫
- 性格：温和、细心
- 标志性动作：紧张时会推眼镜
- 声音风格：温柔女声（如果豆包支持配音）

【角色B】
- 外貌：28岁男性，短发，穿深蓝色卫衣
- 性格：活泼、幽默
- 标志性动作：喜欢挥手
```

**关键：每次生成新场景时，都要带上角色设定，保证一致性。**

### 场景设定模板

```
【场景1：咖啡厅】
- 时间：下午
- 光线：暖色调，窗外阳光
- 环境：木质桌椅，背景有植物

【场景2：街道】
- 时间：傍晚
- 光线：昏黄路灯
- 环境：城市街道，有行人
```

## 三、第二步：写分镜脚本

### 简单模板

```
场景1（5秒）：
  画面：角色A坐在咖啡厅，低头看手机，表情焦虑
  动作：手指滑动屏幕，眉头微皱
  情绪：紧张/期待

场景2（5秒）：
  画面：角色B推门进入咖啡厅
  动作：环顾四周，看到角色A，微笑挥手
  情绪：轻松/友好

场景3（8秒）：
  画面：角色B走到角色A桌前坐下
  动作：两人交谈，角色A表情从焦虑转为安心
  情绪：转折/温暖

场景4（5秒）：
  画面：两人一起走出咖啡厅
  动作：并肩而行，有说有笑
  情绪：愉悦/结束
```

## 四、第三步：分段写提示词

### 提示词公式

```
【场景设定】+ 【角色描述】+ 【具体动作】+ 【镜头/角度】+ 【情绪/氛围】

示例：
"咖啡厅内，下午暖光。25岁女性，黑色长发，穿白色衬衫，坐在窗边低头看手机，
眉头微皱，表情焦虑。中景镜头，侧面角度。温暖略带紧张的氛围。"
```

### 关键词参考

**镜头类型：**
- 特写：面部表情/物品细节
- 中景：半身，适合对话
- 全景：全身，适合动作/环境展示
- 俯视：上帝视角
- 仰视：增强气势

**角度：**
- 正面/侧面/背面
- 45度角（常用，自然）

**运动：**
- 推镜头：靠近主体，强调情绪
- 拉镜头：远离主体，展示环境
- 跟随：角色移动时镜头跟随
- 固定：镜头不动，角色动

**氛围：**
- 温暖、冷淡、紧张、轻松、悲伤、愉悦、神秘

## 五、连贯性关键技巧

### 1. 角色一致性

```
错误示范：
  场景1："一个女孩坐在咖啡厅"
  场景2："一个年轻女性走进咖啡厅"
  → 豆包可能生成完全不同的人

正确示范：
  场景1："25岁女性，黑色长发，戴细框眼镜，穿白色衬衫，坐在咖啡厅"
  场景2："25岁女性，黑色长发，戴细框眼镜，穿白色衬衫，走进咖啡厅"
  → 每次都完整描述，强制一致性
```

### 2. 场景连续性

```
场景1 → 场景2 的过渡：

场景1结尾：角色A看向门口
场景2开头：门打开，角色B进入

技巧：让前一个场景的结尾"指向"下一个场景
```

### 3. 动作衔接

```
场景A：角色抬手准备开门（动作进行中）
场景B：门被推开，角色走进房间（动作继续）

提示词写法：
  场景A："角色A的手正要触碰到门把手，动作进行中"
  场景B："门被推开，角色A走进房间"
```

### 4. 情绪连贯

```
场景1：焦虑 → 场景2：安心

中间需要过渡：
  场景1.5：角色A看到角色B，表情开始变化
  场景2：角色A露出安心的笑容

不要：焦虑直接跳到开心，缺少过渡
```

### 5. 时间/光线连贯

```
场景1：下午，暖光
场景2：傍晚，昏黄

如果时间跳跃，用过渡画面：
  插入一个"夕阳西下"或"路灯亮起"的空镜头
```

## 六、实战示例

### 剧情简述
女孩在咖啡厅等男友，男友迟到，两人见面后和好。

### 分镜脚本

```
【场景1】等待（5秒）
提示词：
"咖啡厅内，下午，暖色调光线。25岁女性，黑色长发，穿白色连衣裙，
独自坐在窗边，低头看手机，偶尔抬头看向门口，表情焦虑。
中景镜头，侧面角度。略带不安的氛围。"

【场景2】迟到（5秒）
提示词：
"城市街道，傍晚，夕阳西下。28岁男性，短发，穿深蓝色卫衣，
快速奔跑，额头有汗水，表情焦急。
全景镜头，跟随拍摄。紧张急促的氛围。"

【场景3】相遇（8秒）
提示词：
"咖啡厅门口，傍晚，暖光。25岁女性站在门口，双手抱胸，表情不满。
28岁男性跑来，喘气，露出歉意的笑容。
中景镜头，双人构图。从紧张转为舒缓的氛围。"

【场景4】和好（6秒）
提示词：
"咖啡厅门口，傍晚。25岁女性和28岁男性面对面站立，
女性表情软化，轻轻笑了，男性伸出手，女性握住。
特写镜头，聚焦两人手部。温暖感人的氛围。"

【场景5】结束（5秒）
提示词：
"咖啡厅外的街道，夜幕降临，路灯亮起。两人并肩走远，背影，
影子拉长。
全景镜头，背影拍摄。温馨浪漫的氛围。"
```

## 七、常见问题

### 问题1：每个场景角色长得不一样

```
解决：
  - 每次提示词都完整描述角色外貌（不要简化）
  - 如果豆包支持，上传参考图保持一致性
  - 用"同一个人"关键词强调
```

### 问题2：场景之间感觉不连贯

```
解决：
  - 检查是否有过渡（动作/情绪/时间）
  - 在剪辑时加转场效果（淡入淡出/叠化）
  - 用背景音乐串联场景
```

### 问题3：生成的内容和预期不符

```
解决：
  - 提示词更具体，减少模糊描述
  - 拆分复杂动作为多个场景
  - 多生成几次，选最接近的
```

### 问题4：场景太短，不够用

```
解决：
  - 豆包单次生成有限，接受这个限制
  - 多生成几个场景拼接
  - 用慢动作/重复画面延长（剪辑技巧）
```

## 八、剪辑拼接技巧

### 工具推荐

- 剪映（手机/电脑，免费，够用）
- CapCut（剪映海外版）
- Premiere Pro（专业）

### 拼接原则

```
1. 动作剪辑：在动作进行中切换
2. 情绪过渡：加淡入淡出/叠化转场
3. 音乐统一：一首背景音乐贯穿
4. 音效补充：脚步声、开门声、环境音
5. 调色统一：所有场景调成一致的色调
```

### 快速工作流

```
1. 豆包生成 → 下载所有视频片段
2. 导入剪映 → 按脚本顺序排列
3. 粗剪 → 删掉不合适的部分
4. 精剪 → 调整过渡、加音乐、加字幕
5. 导出 → 发布
```

## 九、提示词模板（可直接复制）

### 基础模板

```
【场景】：[地点]，[时间]，[光线描述]
【角色】：[年龄]岁[性别]，[发型]，[服装]，[标志性特征]
【动作】：[具体动作描述]，[表情变化]
【镜头】：[镜头类型]，[角度]
【氛围】：[情绪关键词]

合成：
"[场景]。[角色]，[动作]。[镜头]。[氛围]。"
```

### 对话场景模板

```
"室内，白天，自然光。[角色A描述]和[角色B描述]面对面站立，
[角色A动作]，[角色B动作]，两人[交互描述]。
中景镜头，双人构图。[氛围]的氛围。"
```

### 情绪变化模板

```
"[场景]。[角色描述]，[起始情绪]，
然后[触发事件]，表情逐渐变化为[目标情绪]。
特写镜头，聚焦面部。[从起始情绪到目标情绪]的过渡。"
```
