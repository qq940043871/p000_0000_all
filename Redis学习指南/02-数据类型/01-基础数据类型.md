## 一、String（字符串）

### 1.1 概述

```
String是Redis最基础的数据类型

特点:
- 二进制安全，可存储任意数据
- 最大512MB
- 可存储字符串、整数、浮点数、二进制数据
```

### 1.2 基本命令

```bash
SET key value
SET key value EX 60
SET key value PX 60000
SET key value NX
SET key value XX

GET key

DEL key

MSET key1 value1 key2 value2
MGET key1 key2

SETEX key 60 value
SETNX key value

APPEND key suffix
STRLEN key

GETRANGE key 0 5
SETRANGE key 0 newvalue
```

### 1.3 数值操作

```bash
INCR key
INCRBY key 10
INCRBYFLOAT key 2.5

DECR key
DECRBY key 10
```

### 1.4 应用场景

```bash
SET user:1001:name "张三"
GET user:1001:name

INCR article:1001:views
INCRBY video:2001:likes 1

SET session:abc123 "user_data" EX 1800

SET lock:order:1001 "uuid" NX EX 30
```

### 1.5 Java操作

```java
import redis.clients.jedis.Jedis;

Jedis jedis = new Jedis("localhost", 6379);

jedis.set("key", "value");
String value = jedis.get("key");

jedis.setex("key", 60, "value");

jedis.setnx("key", "value");

jedis.incr("counter");
jedis.incrBy("counter", 10);

jedis.mset("k1", "v1", "k2", "v2");
List<String> values = jedis.mget("k1", "k2");
```

## 二、List（列表）

### 2.1 概述

```
List是简单的字符串列表，按插入顺序排序

特点:
- 双向链表结构
- 头尾操作O(1)
- 索引访问O(n)
- 可存储重复元素
```

### 2.2 基本命令

```bash
LPUSH key value1 value2
RPUSH key value1 value2

LPOP key
RPOP key

LLEN key

LRANGE key 0 -1

LINDEX key 0

LSET key 0 newvalue

LREM key 2 value

LTRIM key 0 99
```

### 2.3 阻塞操作

```bash
BLPOP key1 key2 5
BRPOP key1 key2 5

BRPOPLPUSH source destination 5
```

### 2.4 应用场景

```bash
LPUSH queue:task "task1"
RPOP queue:task

LPUSH log:app "log1"
LTRIM log:app 0 999

LPUSH timeline:user:1001 "post:2001"
LRANGE timeline:user:1001 0 9
```

### 2.5 Java操作

```java
jedis.lpush("queue", "task1", "task2");
jedis.rpush("queue", "task3");

String task = jedis.lpop("queue");
String task = jedis.rpop("queue");

List<String> list = jedis.lrange("queue", 0, -1);

long len = jedis.llen("queue");

List<String> tasks = jedis.brpop(5, "queue");
```

## 三、Hash（哈希）

### 3.1 概述

```
Hash是键值对集合，适合存储对象

特点:
- 适合存储对象属性
- 单个Hash可存储2^32-1个字段
- 内存效率高
```

### 3.2 基本命令

```bash
HSET key field value
HGET key field

HMSET key field1 value1 field2 value2
HMGET key field1 field2

HGETALL key

HDEL key field1 field2

HEXISTS key field

HKEYS key
HVALS key

HLEN key

HINCRBY key field 1
HINCRBYFLOAT key field 2.5

HSETNX key field value
```

### 3.3 应用场景

```bash
HSET user:1001 name "张三" age 25 email "test@example.com"
HGET user:1001 name
HGETALL user:1001

HINCRBY article:2001 views 1

HSET product:3001 stock 100
HINCRBY product:3001 stock -1
```

### 3.4 Java操作

```java
jedis.hset("user:1001", "name", "张三");
jedis.hset("user:1001", "age", "25");

Map<String, String> user = new HashMap<>();
user.put("name", "张三");
user.put("age", "25");
jedis.hmset("user:1001", user);

String name = jedis.hget("user:1001", "name");
List<String> values = jedis.hmget("user:1001", "name", "age");

Map<String, String> allFields = jedis.hgetAll("user:1001");

jedis.hincrBy("user:1001", "age", 1);
```

## 四、Set（集合）

### 4.1 概述

```
Set是无序的字符串集合

特点:
- 元素唯一，不重复
- 无序存储
- 支持集合运算
- 添加/删除O(1)
```

### 4.2 基本命令

```bash
SADD key member1 member2

SREM key member1 member2

SMEMBERS key

SISMEMBER key member

SCARD key

SPOP key
SRANDMEMBER key 2
```

### 4.3 集合运算

```bash
SINTER key1 key2
SINTERSTORE dest key1 key2

SUNION key1 key2
SUNIONSTORE dest key1 key2

SDIFF key1 key2
SDIFFSTORE dest key1 key2
```

### 4.4 应用场景

```bash
SADD user:1001:following user:1002 user:1003
SADD user:1002:following user:1001 user:1003

SINTER user:1001:following user:1002:following

SADD tag:java article:1 article:2
SADD tag:redis article:2 article:3

SUNION tag:java tag:redis

SADD lottery:users user1 user2 user3
SRANDMEMBER lottery:users 1
```

### 4.5 Java操作

```java
jedis.sadd("set", "a", "b", "c");

jedis.srem("set", "a");

Set<String> members = jedis.smembers("set");

boolean exists = jedis.sismember("set", "a");

long count = jedis.scard("set");

Set<String> intersection = jedis.sinter("set1", "set2");
Set<String> union = jedis.sunion("set1", "set2");
Set<String> diff = jedis.sdiff("set1", "set2");

String member = jedis.spop("set");
```

## 五、Sorted Set（有序集合）

### 5.1 概述

```
Sorted Set是有序的字符串集合，每个元素关联一个分数

特点:
- 元素唯一
- 按分数排序
- 支持范围查询
- 添加O(log n)
```

### 5.2 基本命令

```bash
ZADD key score1 member1 score2 member2

ZREM key member

ZSCORE key member

ZCARD key

ZRANK key member
ZREVRANK key member

ZRANGE key 0 -1
ZRANGE key 0 -1 WITHSCORES
ZREVRANGE key 0 -1 WITHSCORES

ZRANGEBYSCORE key 0 100
ZRANGEBYSCORE key (0 (100
ZREVRANGEBYSCORE key 100 0

ZINCRBY key 10 member

ZCOUNT key 0 100
```

### 5.3 集合运算

```bash
ZUNIONSTORE dest 2 key1 key2
ZUNIONSTORE dest 2 key1 key2 WEIGHTS 1 2
ZUNIONSTORE dest 2 key1 key2 AGGREGATE SUM

ZINTERSTORE dest 2 key1 key2
```

### 5.4 应用场景

```bash
ZADD leaderboard 100 user1 200 user2 150 user3
ZREVRANGE leaderboard 0 9 WITHSCORES

ZINCRBY leaderboard 50 user1

ZADD delay:queue 1712304000 task1

ZADD realtime:rank 95.5 user1
ZREVRANK realtime:rank user1
```

### 5.5 Java操作

```java
jedis.zadd("zset", 1, "a");
jedis.zadd("zset", 2, "b");

Map<String, Double> members = new HashMap<>();
members.put("c", 3.0);
members.put("d", 4.0);
jedis.zadd("zset", members);

jedis.zrem("zset", "a");

double score = jedis.zscore("zset", "a");

long rank = jedis.zrank("zset", "a");

Set<String> range = jedis.zrange("zset", 0, -1);
Set<Tuple> rangeWithScores = jedis.zrangeWithScores("zset", 0, -1);

Set<String> byScore = jedis.zrangeByScore("zset", 0, 100);

jedis.zincrby("zset", 10, "a");
```

## 六、数据类型对比

| 类型 | 底层结构 | 有序 | 唯一 | 时间复杂度 | 适用场景 |
|------|----------|------|------|------------|----------|
| String | SDS | - | - | O(1) | 缓存、计数器 |
| List | 双向链表 | 是 | 否 | 头尾O(1) | 队列、时间线 |
| Hash | 哈希表 | 否 | 字段唯一 | O(1) | 对象存储 |
| Set | 哈希表/整数集合 | 否 | 是 | O(1) | 标签、社交 |
| Sorted Set | 跳表+哈希表 | 是 | 是 | O(log n) | 排行榜 |
