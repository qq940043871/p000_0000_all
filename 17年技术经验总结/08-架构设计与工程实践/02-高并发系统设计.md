## 一、限流

### 1. 固定窗口

```java
public class FixedWindowRateLimiter {
    private int limit;
    private AtomicInteger count = new AtomicInteger(0);
    private long windowStart = System.currentTimeMillis();
    private long windowSize = 1000;  // 1秒
    
    public boolean tryAcquire() {
        long now = System.currentTimeMillis();
        if (now - windowStart > windowSize) {
            count.set(0);
            windowStart = now;
        }
        return count.incrementAndGet() <= limit;
    }
}
```

### 2. 滑动窗口

```java
public class SlidingWindowRateLimiter {
    private int limit;
    private LinkedList<Long> timestamps = new LinkedList<>();
    private long windowSize = 1000;
    
    public synchronized boolean tryAcquire() {
        long now = System.currentTimeMillis();
        
        // 移除窗口外的请求
        while (!timestamps.isEmpty() && 
               now - timestamps.peek() > windowSize) {
            timestamps.poll();
        }
        
        if (timestamps.size() < limit) {
            timestamps.offer(now);
            return true;
        }
        return false;
    }
}
```

### 3. 令牌桶

```java
public class TokenBucketRateLimiter {
    private int capacity;
    private int rate;  // 每秒生成令牌数
    private AtomicInteger tokens = new AtomicInteger(0);
    private long lastRefillTime = System.currentTimeMillis();
    
    public boolean tryAcquire() {
        refill();
        return tokens.getAndDecrement() > 0;
    }
    
    private void refill() {
        long now = System.currentTimeMillis();
        int newTokens = (int) ((now - lastRefillTime) / 1000.0 * rate);
        if (newTokens > 0) {
            tokens.set(Math.min(capacity, tokens.get() + newTokens));
            lastRefillTime = now;
        }
    }
}
```

### 4. 漏桶

```java
public class LeakyBucketRateLimiter {
    private int capacity;
    private int rate;  // 每秒流出请求数
    private AtomicInteger water = new AtomicInteger(0);
    private long lastLeakTime = System.currentTimeMillis();
    
    public boolean tryAcquire() {
        leak();
        if (water.get() < capacity) {
            water.incrementAndGet();
            return true;
        }
        return false;
    }
    
    private void leak() {
        long now = System.currentTimeMillis();
        int leaked = (int) ((now - lastLeakTime) / 1000.0 * rate);
        if (leaked > 0) {
            water.set(Math.max(0, water.get() - leaked));
            lastLeakTime = now;
        }
    }
}
```

## 三、熔断

### 1. 状态机

```
三种状态：
  Closed（关闭）：
    - 正常请求
    - 失败率超过阈值 → Open

  Open（打开）：
    - 直接拒绝请求
    - 超时后 → Half-Open

  Half-Open（半开）：
    - 允许部分请求
    - 成功 → Closed
    - 失败 → Open
```

### 2. 实现

```java
public class CircuitBreaker {
    private State state = State.CLOSED;
    private int failureCount = 0;
    private int failureThreshold = 10;
    private long openTime = 0;
    private long timeout = 30000;  // 30秒
    
    public boolean allowRequest() {
        switch (state) {
            case CLOSED:
                return true;
            case OPEN:
                if (System.currentTimeMillis() - openTime > timeout) {
                    state = State.HALF_OPEN;
                    return true;
                }
                return false;
            case HALF_OPEN:
                return true;
            default:
                return false;
        }
    }
    
    public void recordSuccess() {
        failureCount = 0;
        state = State.CLOSED;
    }
    
    public void recordFailure() {
        failureCount++;
        if (failureCount >= failureThreshold) {
            state = State.OPEN;
            openTime = System.currentTimeMillis();
        }
    }
}
```

## 四、隔离

### 1. 线程池隔离

```java
@Service
public class OrderService {
    
    private ExecutorService orderPool = Executors.newFixedThreadPool(20);
    private ExecutorService paymentPool = Executors.newFixedThreadPool(10);
    
    public Future<Order> getOrderAsync(Long id) {
        return orderPool.submit(() -> orderClient.getOrder(id));
    }
    
    public Future<Payment> getPaymentAsync(Long id) {
        return paymentPool.submit(() -> paymentClient.getPayment(id));
    }
}
```

### 2. 信号量隔离

```java
public class SemaphoreIsolation {
    private Semaphore semaphore = new Semaphore(10);
    
    public <T> T execute(Supplier<T> supplier) {
        try {
            semaphore.acquire();
            return supplier.get();
        } catch (InterruptedException e) {
            throw new RuntimeException(e);
        } finally {
            semaphore.release();
        }
    }
}
```

## 五、缓存

### 1. 多级缓存

```
请求 → 本地缓存(Caffeine) → Redis → 数据库

本地缓存：
  - 极快（微秒级）
  - 容量有限
  - 一致性问题

Redis缓存：
  - 快（毫秒级）
  - 容量大
  - 集群支持
```

### 2. 缓存策略

```java
@Service
public class CacheService {
    
    @Cacheable(value = "order", key = "#id", cacheManager = "localCacheManager")
    public Order getOrderLocal(Long id) {
        return getOrderFromRedis(id);
    }
    
    @Cacheable(value = "order", key = "#id")
    public Order getOrderFromRedis(Long id) {
        return getOrderFromDB(id);
    }
    
    public Order getOrderFromDB(Long id) {
        return orderRepository.findById(id);
    }
}
```
