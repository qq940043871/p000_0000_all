## 一、缓存策略

### 1. Cache Aside

```
读策略：
  1. 先查缓存
  2. 缓存命中 → 返回
  3. 缓存未命中 → 查数据库 → 写入缓存 → 返回

写策略：
  1. 更新数据库
  2. 删除缓存（不是更新）

优点：读多写少场景性能好
缺点：更新缓存时可能出现不一致
```

### 2. Read Through

```
客户端只查缓存
缓存自动从数据库加载
缓存不存在 → 缓存查数据库 → 返回并写入缓存
```

### 3. Write Through

```
写操作同时更新缓存和数据库
两者同步完成
一致性好，但写入慢
```

### 4. Write Behind

```
写操作先写缓存
缓存异步批量写数据库
性能最好，但可能丢数据
```

## 二、分布式应用

### 1. 分布式锁

```python
import redis
import time
import uuid

class DistributedLock:
    def __init__(self, client, lock_name, timeout=30):
        self.client = client
        self.lock_name = lock_name
        self.timeout = timeout
        self.identifier = str(uuid.uuid4())
    
    def acquire(self):
        end = time.time() + self.timeout
        while time.time() < end:
            if self.client.set(self.lock_name, self.identifier, nx=True, ex=self.timeout):
                return True
            time.sleep(0.001)
        return False
    
    def release(self):
        # Lua脚本保证原子性
        script = """
        if redis.call("get", KEYS[1]) == ARGV[1] then
            return redis.call("del", KEYS[1])
        else
            return 0
        end
        """
        self.client.eval(script, 1, self.lock_name, self.identifier)
```

### 3. 延迟队列

```python
import redis

def add_delay_task(queue, task, delay_seconds):
    execute_time = time.time() + delay_seconds
    score = execute_time
    r.zadd(queue, {json.dumps(task): score})

def consume_delay_task(queue, timeout=5):
    while True:
        now = time.time()
        result = r.zrangebyscore(queue, 0, now, start=0, num=1)
        if not result:
            time.sleep(0.1)
            continue
        task_json = result[0]
        if r.zrem(queue, task_json):
            return json.loads(task_json)
```

## 三、实战优化

### 1. 性能优化

```python
# 1. Pipeline批量操作
pipe = r.pipeline()
for key in keys:
    pipe.get(key)
results = pipe.execute()

# 2. Lua脚本减少网络往返
script = """
local keys = KEYS
local result = {}
for i, key in ipairs(keys) do
    result[i] = redis.call('GET', key)
end
return cjson.encode(result)
"""

# 3. 连接池
pool = redis.ConnectionPool(host='localhost', port=6379, max_connections=10)
client = redis.Redis(connection_pool=pool)

# 4. 集群连接
```

### 2. 可靠性

```python
# 1. 哨兵模式
from redis.sentinel import Sentinel
sentinel = Sentinel([('host1', 26379), ('host2', 26379)], socket_timeout=0.1)
master = sentinel.master_for('mymaster', socket_timeout=0.1)
slave = sentinel.slave_for('mymaster', socket_timeout=0.1)

# 2. 集群模式
from redis.cluster import RedisCluster
rc = RedisCluster(host='10.0.0.1', port=7000)

# 3. 读写分离
def read_from_slave(key):
    return slave.get(key)

def write_to_master(key, value):
    return master.set(key, value)
```
