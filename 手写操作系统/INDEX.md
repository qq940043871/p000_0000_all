## 💻 代码示例

### 最小化内核

```asm
; boot.asm
[bits 16]
org 0x7c00

start:
    mov ax, 0
    mov ds, ax
    mov es, ax
    mov ss, ax
    mov sp, 0x7c00
    
    mov si, msg
    call print_string
    jmp $

print_string:
    lodsb
    cmp al, 0
    je .done
    mov ah, 0x0E
    int 0x10
    jmp print_string
.done:
    ret

msg db "Hello OS!", 0

times 510 - ($ - $$) db 0
dw 0xaa55
```

### 编译和运行

```bash
# 编译
nasm -f bin boot.asm -o boot.img

# 运行
qemu-system-i386 -fda boot.img
```

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone <repo-url>
cd 自制操作系统
```

### 2. 安装依赖
```bash
# Ubuntu/Debian
sudo apt-get install build-essential nasm qemu-system-x86 gdb

# macOS
brew install gcc nasm qemu
```

### 3. 编译
```bash
cd code
make all
```

### 4. 运行
```bash
./run.sh
```

### 5. 调试
```bash
./debug.sh
```

## 📝 学习建议

1. **循序渐进**：从简单到复杂，不要跳过基础
2. **动手实践**：每个模块都要编写代码并测试
3. **理解原理**：不仅要知道怎么做，更要理解为什么
4. **调试技巧**：学会使用GDB和QEMU进行调试
5. **代码阅读**：阅读Linux等开源OS的源码
6. **文档记录**：记录学习过程和遇到的问题
