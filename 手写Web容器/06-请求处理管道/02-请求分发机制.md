# 请求分发机制

> 模块：请求处理管道
> 更新时间：2026-03-29

---

## 一、RequestDispatcher

```java
public interface RequestDispatcher {
    void forward(ServletRequest request, ServletResponse response) 
        throws ServletException, IOException;
    
    void include(ServletRequest request, ServletResponse response) 
        throws ServletException, IOException;
}
```

---

## 二、实现类

```java
public class ApplicationRequestDispatcher implements RequestDispatcher {
    private Wrapper wrapper;
    private String path;
    
    public ApplicationRequestDispatcher(Wrapper wrapper) {
        this.wrapper = wrapper;
    }
    
    public ApplicationRequestDispatcher(Wrapper wrapper, String path) {
        this.wrapper = wrapper;
        this.path = path;
    }
    
    @Override
    public void forward(ServletRequest request, ServletResponse response) 
        throws ServletException, IOException {
        
        // 清空之前输出的内容
        if (response.isCommitted()) {
            throw new IllegalStateException("Response already committed");
        }
        
        response.resetBuffer();
        
        // 更新请求路径
        if (path != null) {
            ((HttpServletRequest) request).setAttribute("javax.servlet.forward.path_info", path);
        }
        
        // 调用目标Servlet
        Servlet servlet = wrapper.allocate();
        servlet.service(request, response);
        wrapper.deallocate(servlet);
    }
    
    @Override
    public void include(ServletRequest request, ServletResponse response) 
        throws ServletException, IOException {
        
        // 保存原路径
        if (path != null) {
            ((HttpServletRequest) request).setAttribute("javax.servlet.include.path_info", path);
        }
        
        // 调用目标Servlet（不清空响应）
        Servlet servlet = wrapper.allocate();
        servlet.service(request, response);
        wrapper.deallocate(servlet);
    }
}
```
