## 一、连接池设计

```
连接池作用：
  - 限制最大连接数
  - 复用连接资源
  - 防止资源耗尽

核心参数：
  - maxConnections: 最大连接数（默认10000）
  - acceptCount: 等待队列长度（默认100）
  - connectionTimeout: 连接超时（默认20000ms）
```

## 二、实现代码

```java
public class ConnectionPool {
    private AtomicInteger activeConnections = new AtomicInteger(0);
    private Semaphore connectionSemaphore;
    private BlockingQueue<Socket> waitingQueue;
    private int maxConnections = 10000;
    private int acceptCount = 100;
    
    public ConnectionPool() {
        this.connectionSemaphore = new Semaphore(maxConnections);
        this.waitingQueue = new ArrayBlockingQueue<>(acceptCount);
    }
    
    public boolean tryAcquire() {
        if (connectionSemaphore.tryAcquire()) {
            activeConnections.incrementAndGet();
            return true;
        }
        return false;
    }
    
    public void release() {
        connectionSemaphore.release();
        activeConnections.decrementAndGet();
    }
    
    public int getActiveConnections() {
        return activeConnections.get();
    }
    
    public boolean isMaxConnectionsReached() {
        return activeConnections.get() >= maxConnections;
    }
}
```

## 三、限流机制

```java
public class LimitLatch {
    private final AtomicInteger count = new AtomicInteger(0);
    private volatile int limit;
    private final Object lock = new Object();
    
    public LimitLatch(int limit) {
        this.limit = limit;
    }
    
    public void countUpOrAwait() throws InterruptedException {
        synchronized (lock) {
            while (count.get() >= limit) {
                lock.wait();
            }
            count.incrementAndGet();
        }
    }
    
    public void countDown() {
        synchronized (lock) {
            count.decrementAndGet();
            lock.notifyAll();
        }
    }
    
    public int getCount() {
        return count.get();
    }
}
```
