## 一、优雅停机流程

```
1. 停止接收新连接
2. 等待现有请求完成
3. 关闭线程池
4. 释放资源
5. 完全停止
```

---

## 二、实现代码

```java
public class GracefulShutdown {
    private ExecutorService threadPool;
    private volatile boolean accepting = true;
    private AtomicInteger activeRequests = new AtomicInteger(0);
    private long shutdownTimeout = 30000;  // 30秒
    
    public void shutdown() throws InterruptedException {
        System.out.println("Starting graceful shutdown...");
        
        // 1. 停止接收新连接
        accepting = false;
        System.out.println("Stopped accepting new connections");
        
        // 2. 等待现有请求完成
        long startTime = System.currentTimeMillis();
        while (activeRequests.get() > 0) {
            long elapsed = System.currentTimeMillis() - startTime;
            if (elapsed > shutdownTimeout) {
                System.out.println("Shutdown timeout, forcing close");
                break;
            }
            
            System.out.println("Waiting for " + activeRequests.get() + " active requests");
            Thread.sleep(1000);
        }
        
        // 3. 关闭线程池
        threadPool.shutdown();
        if (!threadPool.awaitTermination(10, TimeUnit.SECONDS)) {
            threadPool.shutdownNow();
        }
        
        System.out.println("Graceful shutdown completed");
    }
    
    public void beforeRequest() {
        if (!accepting) {
            throw new IllegalStateException("Server is shutting down");
        }
        activeRequests.incrementAndGet();
    }
    
    public void afterRequest() {
        activeRequests.decrementAndGet();
    }
}
```

## 三、Connector集成

```java
public class Connector {
    private GracefulShutdown gracefulShutdown;
    
    public void handleRequest(Socket socket) {
        try {
            gracefulShutdown.beforeRequest();
            
            // 处理请求
            processRequest(socket);
            
        } finally {
            gracefulShutdown.afterRequest();
        }
    }
}
```
