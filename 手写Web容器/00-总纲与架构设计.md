## 技术选型

| 模块 | 技术选择 | 说明 |
|------|---------|------|
| 网络IO | Java NIO | 高并发，非阻塞 |
| 线程模型 | Reactor模式 | 主从Reactor |
| HTTP解析 | 手写状态机 | 高性能解析 |
| 配置解析 | DOM/SAX | XML解析 |
| 类加载 | 自定义ClassLoader | 应用隔离 |
| 日志 | SLF4J + Logback | 标准日志 |
| 构建工具 | Maven | 依赖管理 |
