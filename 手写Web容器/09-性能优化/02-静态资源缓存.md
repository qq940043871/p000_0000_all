## 一、缓存策略

```
1. 内存缓存
   - 小文件直接缓存在内存
   - 减少磁盘IO

2. 浏览器缓存
   - 设置Cache-Control头
   - 减少网络传输

3. CDN缓存
   - 分布式缓存
   - 加速访问
```

## 二、实现代码

```java
public class StaticResourceCache {
    private Map<String, CachedResource> cache = new ConcurrentHashMap<>();
    private long maxCacheSize = 100 * 1024 * 1024;  // 100MB
    private long currentSize = 0;
    
    public byte[] getResource(String path) throws IOException {
        CachedResource cached = cache.get(path);
        
        if (cached != null && !cached.isExpired()) {
            return cached.getData();
        }
        
        // 从磁盘读取
        File file = new File(path);
        byte[] data = Files.readAllBytes(file.toPath());
        
        // 缓存
        if (data.length < 1024 * 1024) {  // 只缓存<1MB的文件
            if (currentSize + data.length < maxCacheSize) {
                cache.put(path, new CachedResource(data, file.lastModified()));
                currentSize += data.length;
            }
        }
        
        return data;
    }
    
    private static class CachedResource {
        private byte[] data;
        private long lastModified;
        private long cachedTime;
        
        public CachedResource(byte[] data, long lastModified) {
            this.data = data;
            this.lastModified = lastModified;
            this.cachedTime = System.currentTimeMillis();
        }
        
        public boolean isExpired() {
            // 5分钟过期
            return System.currentTimeMillis() - cachedTime > 5 * 60 * 1000;
        }
        
        public byte[] getData() {
            return data;
        }
    }
}
```

## 三、缓存头设置

```java
public class CacheHeaderUtil {
    
    public static void setCacheHeaders(HttpResponse response, File file) {
        // 设置Last-Modified
        long lastModified = file.lastModified();
        response.setHeader("Last-Modified", formatDate(new Date(lastModified)));
        
        // 设置ETag
        String etag = "\"" + file.length() + "-" + lastModified + "\"";
        response.setHeader("ETag", etag);
        
        // 设置Cache-Control
        response.setHeader("Cache-Control", "public, max-age=3600");
        
        // 设置Expires
        long expiresTime = System.currentTimeMillis() + 3600 * 1000;
        response.setHeader("Expires", formatDate(new Date(expiresTime)));
    }
    
    public static boolean checkNotModified(HttpRequest request, File file) {
        String ifModifiedSince = request.getHeader("If-Modified-Since");
        if (ifModifiedSince != null) {
            long clientTime = parseDate(ifModifiedSince);
            if (clientTime >= file.lastModified()) {
                return true;
            }
        }
        
        String ifNoneMatch = request.getHeader("If-None-Match");
        if (ifNoneMatch != null) {
            String etag = "\"" + file.length() + "-" + file.lastModified() + "\"";
            if (ifNoneMatch.equals(etag)) {
                return true;
            }
        }
        
        return false;
    }
}
```
