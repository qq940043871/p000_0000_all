## 二、虚拟内存

### 虚拟内存原理

```
虚拟地址空间：程序使用的逻辑地址空间
物理地址空间：实际硬件内存

虚拟内存优势：
1. 进程隔离，互不干扰
2. 简化内存分配
3. 支持内存超量使用(Swap)
4. 共享代码库
```

### 分页机制

```bash
# 页大小
getconf PAGESIZE                    # 通常是4KB

# 查看页表
cat /proc/meminfo | grep -i page
# AnonPages:      用户态匿名页
# PageTables:     页表占用
# Slab:           内核Slab分配器
```

## 三、内存分配

### 用户态分配

```c
// 堆内存分配
void *malloc(size_t size);          // 分配
void *calloc(size_t n, size_t size); // 分配并清零
void *realloc(void *ptr, size_t size); // 重新分配
void free(void *ptr);                // 释放

// 底层分配
void *sbrk(intptr_t increment);     // 调整堆大小
void *mmap(void *addr, size_t len, int prot, int flags, int fd, off_t offset);
```

### 内存分配器(glibc)

```
内存分配器优化：减少系统调用，提高分配效率

Arena: 线程分配区
Chunk: 内存块
Bin:   空闲链表

分配策略：
1. Fast bin: 小块(<=64B)，不合并
2. Small bin: 中块(<=1024B)，FIFO
3. Large bin: 大块，Best-fit
4. Unsorted bin: 刚释放的chunk
```

## 四、Swap交换空间

### Swap原理

```
物理内存不足 → 页面置换 → Swap

页面置换算法：
- LRU (Least Recently Used)
- CLOCK-Pro
```

### Swap配置

```bash
# 查看Swap
swapon -s                        # 显示Swap设备
free -h                          # 显示内存和Swap

# 创建Swap文件
dd if=/dev/zero of=/swapfile bs=1M count=2048
mkswap /swapfile
chmod 600 /swapfile
swapon /swapfile

# 永久启用
echo "/swapfile none swap sw 0 0" >> /etc/fstab

# Swap使用监控
cat /proc/meminfo | grep -i swap
vmstat 1
```

## 五、内存映射

### /proc/meminfo详解

```bash
cat /proc/meminfo

# 关键指标说明：
# MemTotal:       物理内存总量
# MemFree:        空闲内存
# MemAvailable:   可用内存(含缓存)
# Buffers:        块设备缓存
# Cached:         文件系统缓存
# SwapCached:     Swap中的缓存页
# Active:         活跃内存
# Inactive:       非活跃内存
# AnonPages:      用户态匿名页
# Mapped:         映射的文件页
# Shmem:          共享内存
# Slab:           内核Slab对象
# SReclaimable:   可回收的Slab
# SUnreclaim:     不可回收Slab
# KernelStack:    内核栈
# PageTables:     页表内存
# NFS_Unstable:   NFS不稳定页
# SwapTotal:      Swap总量
# SwapFree:       Swap空闲
# Dirty:          脏页
# Writeback:      正在回写的页
```

### 内存计算公式

```
可用内存 = MemFree + Cached + Buffers + SReclaimable

实际可用 = MemAvailable (内核计算好的)

用户使用 = RSS (Resident Set Size) = 实际占用物理内存

虚拟内存 = VSZ (Virtual Size) = 分配的虚拟空间
```

## 六、内存回收

### 页面回收机制

```bash
# 内存回收触发
# 1. 水位线触发
#    - min_free_kbytes: 最小空闲
#    - low_free_kbytes: 低水位
#    - high_free_kbytes: 高水位

# 2. 查看水位
cat /proc/sys/vm/min_free_kbytes

# 3. 修改水位(需要root)
sysctl -w vm.min_free_kbytes=65536
```

### OOM Killer

```bash
# /proc/vmstat中的OOM统计
cat /proc/vmstat | grep -i oom

# oom_kill计数
cat /proc/vmstat | grep oom_kill

# 查看OOM分数
# 每个进程有oom_score，范围-17到1000
# 分数越高，越容易被OOM Killer杀死
cat /proc/PID/oom_score

# 调整OOM偏好
# oom_score_adj: 手动调整
echo -1000 > /proc/PID/oom_score_adj  # 禁止杀死
echo 500 > /proc/PID/oom_score_adj    # 更容易杀死
```
