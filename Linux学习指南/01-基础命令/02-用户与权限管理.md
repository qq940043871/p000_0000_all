## 一、用户管理

### 用户相关文件

```
/etc/passwd    # 用户信息
/etc/shadow    # 密码信息（加密）
/etc/group     # 组信息
/etc/gshadow   # 组密码（加密）
/home/username # 用户主目录
```

### 用户操作

```bash
# ====== 创建用户 ======
useradd username                # 创建用户
useradd -m username             # 创建并创建主目录
useradd -m -s /bin/bash username  # 指定shell
useradd -g groupname username   # 指定主组
useradd -G group1,group2 username  # 指定附加组
useradd -u 1000 username        # 指定UID
useradd -e 2026-12-31 username  # 设置过期时间
useradd -r username              # 创建系统用户

# ====== 修改用户 ======
usermod -l newname oldname      # 改名
usermod -g groupname username   # 修改主组
usermod -aG groupname username  # 添加附加组
usermod -s /bin/zsh username    # 修改shell
usermod -L username             # 锁定用户
usermod -U username             # 解锁用户
usermod -e "" username          # 取消过期

# ====== 删除用户 ======
userdel username                # 删除用户（保留主目录）
userdel -r username            # 删除用户及主目录

# ====== 查看用户 ======
id username                     # 查看用户信息
whoami                          # 当前用户
who                             # 登录用户
w                               # 详细登录信息
last                            # 登录历史
lastlog                         # 所有用户最后登录
```

## 二、组管理

```bash
# ====== 创建组 ======
groupadd groupname              # 创建组
groupadd -g 1000 groupname     # 指定GID

# ====== 修改组 ======
groupmod -n newname oldname    # 改名
groupmod -g 1000 groupname     # 修改GID

# ====== 删除组 ======
groupdel groupname             # 删除组

# ====== 查看组 ======
groups username                 # 用户所属组
cat /etc/group                  # 所有组
```

## 三、密码管理

```bash
# ====== 设置密码 ======
passwd                          # 设置当前用户密码
passwd username                 # 设置指定用户密码
passwd -l username              # 锁定密码
passwd -u username              # 解锁密码
passwd -d username              # 删除密码（无密码登录）
passwd -e username              # 强制下次登录改密码
passwd -n 7 username            # 最小使用天数
passwd -x 90 username           # 最大使用天数
passwd -w 7 username            # 提前警告天数

# ====== 查看密码状态 ======
chage -l username               # 查看密码策略
```

## 四、sudo权限

```bash
# ====== sudo配置 ======
visudo                          # 编辑sudoers文件

# 示例：授权用户执行所有命令
username ALL=(ALL) ALL

# 示例：授权用户无密码sudo
username ALL=(ALL) NOPASSWD: ALL

# 示例：授权执行特定命令
username ALL=(ALL) /usr/bin/systemctl, /usr/bin/journalctl

# 示例：授权用户组
%wheel ALL=(ALL) ALL
```

## 五、su与身份切换

```bash
# ====== su ======
su - username                   # 切换用户（加载环境）
su username                     # 切换用户（不加载环境）
su -                           # 切换到root
su -c "command" username       # 以其他用户执行命令

# ====== runuser ======
runuser - username -c "command"  # 不启动shell执行
runuser -l username -c "command"  # 登录shell执行
```

## 六、权限深入理解

### 权限位

```
-rwxr-xr-x 1 user group 4096 Jan 29 10:00 file.txt
││││││ │  │    │     │        │
│││││││ │  │    │     │        └── 文件名
│││││││ │  │    │     └────────── 修改时间
│││││││ │  │    └─────────────── 所属组
│││││││ │  └────────────────── 链接数
│││││││ └───────────────────── 所属用户
││││││└────────────────────── 其他用户权限
│││││└─────────────────────── 组用户权限
││││└───────────────────────── 文件所有者权限
│││└────────────────────────── 目录标志(d/-/l)
││└─────────────────────────── 特殊权限
└──────────────────────────────── 文件类型
```

### 特殊权限

```bash
# SETUID (4000): 执行时以文件所有者身份运行
chmod u+s file                  # 可执行文件提权
# 示例: /usr/bin/passwd 有SETUID，任何用户可以修改自己的密码

# SETGID (2000): 执行时以文件所属组身份运行
chmod g+s folder                # 目录内文件继承目录组

# Sticky Bit (1000): 目录内文件只能被创建者删除
chmod +t /tmp                   # /tmp目录
```

## 七、ACL访问控制

```bash
# ====== 查看ACL ======
getfacl file.txt                # 查看文件ACL

# ====== 设置ACL ======
setfacl -m u:username:r file.txt       # 给用户设置权限
setfacl -m g:groupname:rw file.txt     # 给组设置权限
setfacl -m o::r file.txt               # 设置其他人权限
setfacl -m m::rwx file.txt             # 设置最大权限掩码

# ====== 默认ACL ======
setfacl -m d:u:username:rw folder/    # 目录内新建文件默认权限

# ====== 删除ACL ======
setfacl -x u:username file.txt         # 删除用户ACL
setfacl -b file.txt                    # 删除所有ACL
```
