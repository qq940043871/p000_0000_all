## 二、程序计数器

```java
public class PCRegister {
    
    private long pc;  // 当前执行的字节码地址
    
    // 每个线程都有自己的程序计数器
    private Thread thread;
    
    public void setPC(long pc) {
        this.pc = pc;
    }
    
    public long getPC() {
        return pc;
    }
    
    // 执行下一条指令
    public void increment() {
        pc++;
    }
    
    // 跳转
    public void jump(int offset) {
        pc += offset;
    }
}
```

## 三、虚拟机栈

```java
public class JVMStack {
    
    private int maxSize;  // 栈最大容量
    private Frame[] frames;  // 栈帧数组
    private int top;  // 栈顶指针
    
    public JVMStack(int maxSize) {
        this.maxSize = maxSize;
        this.frames = new Frame[maxSize];
        this.top = -1;
    }
    
    // 入栈
    public void push(Frame frame) {
        if (top >= maxSize - 1) {
            throw new StackOverflowError();
        }
        frames[++top] = frame;
    }
    
    // 出栈
    public Frame pop() {
        if (top < 0) {
            throw new IllegalStateException("Stack is empty");
        }
        return frames[top--];
    }
    
    // 获取栈顶
    public Frame peek() {
        if (top < 0) {
            return null;
        }
        return frames[top];
    }
    
    public boolean isEmpty() {
        return top < 0;
    }
}

// 栈帧
public class Frame {
    
    // 局部变量表
    private LocalVariables localVariables;
    
    // 操作数栈
    private OperandStack operandStack;
    
    // 所属方法
    private Method method;
    
    // 返回地址
    private int returnAddress;
    
    // 指向下一个栈帧（用于调试）
    private Frame nextFrame;
}
```

## 四、局部变量表

```java
public class LocalVariables {
    
    private Object[] slots;
    private int maxLocals;
    
    public LocalVariables(int maxLocals) {
        this.maxLocals = maxLocals;
        this.slots = new Object[maxLocals];
    }
    
    public void setInt(int index, int value) {
        slots[index] = value;
    }
    
    public int getInt(int index) {
        Object value = slots[index];
        if (value == null) return 0;
        return (Integer) value;
    }
    
    public void setLong(int index, long value) {
        slots[index] = value;
        slots[index + 1] = SlotsHelper.LONG_MARKER; // 占位
    }
    
    public long getLong(int index) {
        return (Long) slots[index];
    }
    
    public void setRef(int index, Object ref) {
        slots[index] = ref;
    }
    
    public Object getRef(int index) {
        return slots[index];
    }
}
```

## 五、操作数栈

```java
public class OperandStack {
    
    private int maxSize;
    private Object[] slots;
    private int top;
    
    public OperandStack(int maxSize) {
        this.maxSize = maxSize;
        this.slots = new Object[maxSize];
        this.top = -1;
    }
    
    // 入栈
    public void pushInt(int value) {
        slots[++top] = value;
    }
    
    public void pushLong(long value) {
        slots[++top] = value;
        slots[++top] = SlotsHelper.LONG_MARKER;
    }
    
    public void pushRef(Object ref) {
        slots[++top] = ref;
    }
    
    // 出栈
    public int popInt() {
        return (Integer) slots[top--];
    }
    
    public long popLong() {
        top--; // 跳过占位符
        return (Long) slots[top--];
    }
    
    public Object popRef() {
        return slots[top--];
    }
    
    // 查看栈顶
    public int peekInt() {
        return (Integer) slots[top];
    }
    
    public Object peekRef() {
        return slots[top];
    }
    
    public boolean isEmpty() {
        return top < 0;
    }
    
    public int size() {
        return top + 1;
    }
}
```

```java
    private long used;
    
    // GC相关

    public Object allocate(long bytes) {
        if (used + bytes > size) {
```

## 七、方法区

```java
public class MethodArea {
    
    // 类信息
    private Map<String, Class<?>> classes = new HashMap<>();
    
    // 方法字节码
    private Map<String, MethodInfo> methods = new HashMap<>();
    
    // 运行时常量池
    private Map<String, Object> constantPool = new HashMap<>();
    
    // 字符串常量
    private Map<String, String> stringConstants = new HashMap<>();
    
    public void defineClass(String name, ClassFile classFile) {
        Class<?> clazz = new Class(classFile);
        classes.put(name, clazz);
    }
    
    public Class<?> findClass(String name) {
        return classes.get(name);
    }
    
    public MethodInfo findMethod(String className, String methodName) {
        return methods.get(className + "." + methodName);
    }
}
```

## 八、总结

运行时数据区是JVM内存管理的核心，理解各个区域的作用对于排查OOM等问题至关重要。

**学习要点**：
1. 理解各数据区的作用
2. 掌握栈帧的结构
3. 理解局部变量表和操作数栈
4. 理解堆和方法区的区别
