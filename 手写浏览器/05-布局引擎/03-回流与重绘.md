# 回流与重绘

> 模块：布局引擎
> 更新时间：2026-03-29

---

## 一、渲染树到几何信息

```
渲染流程：
  DOM树 → 渲染树 → 布局 → 绘制
  
回流（Reflow/Layout）：
  - 几何属性改变时触发
  - 重新计算布局信息
  - 成本高
  
重绘（Repaint）：
  - 外观改变但几何不变
  - 只需重新绘制
  - 成本相对低
```

## 二、触发回流的属性

```cpp
class LayoutInvalidator {
public:
    // 触发回流的属性
    static std::set<std::string> reflowProperties = {
        "width", "height",
        "margin", "margin-top", "margin-right", "margin-bottom", "margin-left",
        "padding", "padding-top", "padding-right", "padding-bottom", "padding-left",
        "border", "border-width", "border-style", "border-color",
        "display", "position", "top", "right", "bottom", "left",
        "float", "clear", "overflow", "transform",
        "font-size", "font-family", "font-weight",
        "line-height", "text-align", "vertical-align",
        "flex-basis", "flex-grow", "flex-shrink", "align-self"
    };
    
    // 仅触发重绘的属性
    static std::set<std::string> repaintOnlyProperties = {
        "color", "background", "background-color", "background-image",
        "opacity", "visibility", "border-radius",
        "box-shadow", "text-shadow", "outline"
    };
    
    void onStyleChange(DOMElement* element, const std::string& property) {
        if (reflowProperties.count(property)) {
            invalidateReflow(element);
        } else if (repaintOnlyProperties.count(property)) {
            invalidateRepaint(element);
        }
    }
    
private:
    void invalidateReflow(DOMElement* element) {
        // 标记需要回流
        element->setNeedsReflow(true);
        
        // 向上标记所有祖先
        for (DOMElement* parent = element->parentElement(); 
             parent != nullptr; 
             parent = parent->parentElement()) {
            // 如果是position:fixed，不影响祖先布局
            if (parent->style.getProperty("position") == "fixed") {
                break;
            }
        }
        
        // 批量处理
        scheduleReflow();
    }
};
```

## 三、批量更新优化

```cpp
class LayoutScheduler {
private:
    bool needsReflow = false;
    DOMElement* root = nullptr;
    
public:
    void scheduleReflow() {
        if (!needsReflow) {
            needsReflow = true;
            postTask(ReflowTask);
        }
    }
    
    void executeReflow() {
        if (!needsReflow) return;
        
        // 分批处理
        if (root) {
            // 自顶向下计算布局
            reflowSubtree(root);
        }
        
        needsReflow = false;
    }
    
    void reflowSubtree(DOMElement* element) {
        if (element->needsReflow()) {
            // 计算当前元素布局
            LayoutBox* box = element->getLayoutBox();
            layoutBox(box, element->containingWidth());
            
            element->setNeedsReflow(false);
        }
        
        // 递归处理子元素
        for (DOMElement* child = element->firstElementChild(); 
             child != nullptr; 
             child = child->nextElementSibling()) {
            reflowSubtree(child);
        }
    }
};
```
