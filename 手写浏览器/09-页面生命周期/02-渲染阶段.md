# 渲染阶段

> 模块：页面生命周期
> 更新时间：2026-03-29

---

## 一、渲染时机

```
触发渲染的时机：
  1. requestAnimationFrame回调
  2. DOM改动（取决于浏览器调度）
  3. CSS改动（取决于浏览器调度）
  4. scroll/resize事件

渲染步骤：
  1. 更新样式
  2. 计算布局
  3. 绘制
  4. 合成
```

---

## 二、C++实现

```cpp
class RenderingScheduler {
public:
    void requestAnimationFrame(std::function<void()> callback) {
        animationCallbacks.push_back(callback);
        scheduleRendering();
    }
    
    void scheduleRendering() {
        if (!needsRendering_) {
            needsRendering_ = true;
            compositor->scheduleFrame();
        }
    }
    
    void frame() {
        if (!needsRendering_) return;
        
        // 1. 更新样式
        styleResolver.update();
        
        // 2. 计算布局
        if (needsLayout_) {
            layoutEngine.layout();
            needsLayout_ = false;
        }
        
        // 3. 执行动画回调
        for (auto& callback : animationCallbacks) {
            callback();
        }
        animationCallbacks.clear();
        
        // 4. 绘制
        paint();
        
        // 5. 合成
        compositor->composite();
        
        needsRendering_ = false;
    }
    
private:
    bool needsRendering_ = false;
    bool needsLayout_ = false;
    std::vector<std::function<void()>> animationCallbacks;
    StyleResolver* styleResolver;
    LayoutEngine* layoutEngine;
    Compositor* compositor;
};
```
