# 页面加载流程

> 模块：页面生命周期
> 更新时间：2026-03-29

---

## 一、完整加载流程

```
1. URL输入 → 2. DNS解析 → 3. TCP连接 → 4. HTTP请求
         ↓
5. 响应处理 → 6. HTML解析 → 7. DOM构建 → 8. CSS解析
         ↓
9. CSSOM构建 → 10. JS执行 → 11. 布局计算 → 12. 渲染
```

---

## 二、C++实现

```cpp
class PageLoader {
public:
    void load(const URL& url) {
        // 1. DNS解析
        std::vector<IPAddress> ips = dnsResolver.resolve(url.host());
        
        // 2. TCP连接
        TcpConnection* conn = tcpPool.getConnection(
            ips[0], url.port() ? url.port() : 80);
        
        // 3. 发送HTTP请求
        HttpRequest request;
        request.method = "GET";
        request.url = url.path();
        request.headers["Host"] = url.host();
        conn->send(request.serialize());
        
        // 4. 接收响应
        std::string response = conn->receive();
        
        // 5. 解析响应
        HttpResponse httpResponse;
        httpResponse.parse(response);
        
        // 6. 更新状态
        updateLoadingState(LOADING);
        
        // 7. 解析HTML
        std::string html = httpResponse.body;
        std::vector<HTMLToken> tokens = htmlTokenizer.tokenize(html);
        DOMNode* document = treeBuilder.build(tokens);
        
        // 8. 触发DOMContentLoaded
        dispatchEvent("DOMContentLoaded");
        
        // 9. 等待资源加载
        loadResources(document);
        
        // 10. 触发load事件
        dispatchEvent("load");
        
        updateLoadingState(COMPLETE);
    }
    
private:
    void loadResources(DOMNode* document) {
        // 收集所有资源
        std::vector<ResourceRequest> resources;
        collectResources(document, resources);
        
        // 按优先级排序
        sortByPriority(resources);
        
        // 并发下载
        for (auto& req : resources) {
            resourceScheduler.schedule(&req);
        }
    }
};
```
