## 一、关键路径

```
关键渲染路径（Critical Rendering Path）：
  1. HTML → DOM
  2. CSS → CSSOM
  3. DOM + CSSOM → Render Tree
  4. Render Tree → Layout
  5. Layout → Paint
  
优化目标：
  - 最小化关键资源
  - 减少关键路径长度
  - 减少关键字节数
```

## 二、资源优先级

```cpp
enum class ResourcePriority {
    kVeryHigh,  // HTML
    kHigh,      // CSS, 同步JS
    kMedium,    // 异步JS, prefetch
    kLow,       // 图片, 异步JS
    kVeryLow    // preload, 预取
};

class ResourcePriorityCalculator {
public:
    ResourcePriority calculatePriority(ResourceRequest& request) {
        // 1. HTML文档
        if (request.type() == ResourceType::kMainFrame) {
            return ResourcePriority::kVeryHigh;
        }
        
        // 2. CSS
        if (request.type() == ResourceType::kCSS) {
            return ResourcePriority::kHigh;
        }
        
        // 3. 同步JS
        if (request.type() == ResourceType::kScript && 
            !request.defer() && !request.async()) {
            return ResourcePriority::kHigh;
        }
        
        // 4. 图片
        if (request.type() == ResourceType::kImage) {
            return ResourcePriority::kLow;
        }
        
        return ResourcePriority::kMedium;
    }
};
```
