## 一、站点隔离

```
Site Isolation：
  - 不同源的页面在不同进程中
  - 防止Spectre/Meltdown攻击
  - 增强安全边界

隔离策略：
  - OpaqueResponseBlock：跨域阻止
  - Cross-Origin-Opener-Policy
  - Cross-Origin-Embedder-Policy
```

## 二、沙箱模型

```cpp
class Sandbox {
public:
    void initialize() {
        // 设置进程沙箱
        #ifdef WIN32
        SetProcessMitigationPolicy(
            PROCESS_MITIGATION_SYSTEM_CALL_FILTER_POLICY,
            &policy,
            sizeof(policy));
        #endif
        
        // 禁用危险权限
        disableCapabilities();
    }
    
private:
    void disableCapabilities() {
        // 删除特权
        setuid(getuid());
        
        // 限制文件系统访问
        chroot("/sandbox/root");
        
        // 限制网络访问
        // ...
    }
};
```
