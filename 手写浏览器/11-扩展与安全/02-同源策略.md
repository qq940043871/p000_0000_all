## 一、源的定义

```
源（Origin）= 协议 + 域名 + 端口

示例：
  https://example.com:443 → https://example.com:443
  http://example.com → https://example.com（不同协议）
  https://sub.example.com → https://example.com（不同子域）
  https://example.com:8080 → https://example.com（不同端口）
```

---

## 二、跨域限制

```
限制项：
  - Cookie无法跨域读写
  - DOM无法跨域访问
  - AJAX请求无法跨域
  
例外：
  - <script src="...">（JSONP）
  - <link rel="stylesheet">
  - <img>、<video>等媒体标签
  - CORS（Cross-Origin Resource Sharing）
```

## 三、CORS机制

```cpp
class CORSValidator {
public:
    bool validateCrossOriginRequest(const HTTPRequest& request,
                                   const HTTPResponse& response) {
        std::string origin = request.getHeader("Origin");
        if (origin.empty()) return true;
        
        std::string allowOrigin = 
            response.getHeader("Access-Control-Allow-Origin");
        
        if (allowOrigin == "*" || allowOrigin == origin) {
            return true;
        }
        
        return false;
    }
    
    void handlePreflightRequest(const HTTPRequest& request) {
        std::string origin = request.getHeader("Origin");
        std::string method = request.getHeader("Access-Control-Request-Method");
        
        HTTPResponse response;
        response.setHeader("Access-Control-Allow-Origin", origin);
        response.setHeader("Access-Control-Allow-Methods", method);
        response.setHeader("Access-Control-Allow-Headers", 
            request.getHeader("Access-Control-Request-Headers"));
        response.setHeader("Access-Control-Max-Age", "86400");
        
        sendResponse(response);
    }
};
```
